概述

Native SDK 使用委託來響應用戶操作和視頻播放事件。這生命週期事件委託方法提供事件來通知播放狀態的變化。

iOS框架

您的 iOS 應用程序是自定義代碼、本機 SDK 框架和系統框架的組合。這些框架使用模型-視圖-控制器設計模式以及委託來為您的應用程序提供功能。
