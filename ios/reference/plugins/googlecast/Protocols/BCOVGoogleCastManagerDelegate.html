– 未找到合適的來源

當找不到合適的源時調用此方法。

- (void)suitableSourceNotFound

討論

這BCOVGoogleCastManager將查找 HLS v3 的 HTTPS 版本，然後檢查 DASH，最後是 MP4 源。如果未找到 HTTPS 版本，它將檢查相同源類型的 HTTP 版本。如果未找到 HTTP 版本，則將調用此方法。

– 使用SourceFromSources：

當需要在轉換之前解析源時調用此方法。

- (BCOVSource *_Nonnull)useSourceFromSources:(NSArray<BCOVSource*> *_Nonnull)來源

討論

您可以使用此方法指定要與 Cast Receiver 一起使用的源。如果不實現這個方法， BCOVGoogleCastManager將自行執行源選擇（請參閱未找到合適的來源想要查詢更多的信息）。

–  shouldCast視頻：

這個方法在調用之後BCOVGoogleCastManager已收到新的播放會話。

- (BOOL)shouldCastVideo:(BCOVVideo *_Nonnull)視頻

討論

什麼時候BCOVGoogleCastManager從播放會話接收新的 BCOVVideo 它將嘗試將其與之前的 BCOVVideo（如果有）進行比較，以防止投射會話使用相同的視頻重新啟動。您可以使用此方法覆蓋內部檢查，以便您可以控制是否應該投射當前視頻。
