    布萊特灣原生 SDK 功能

    Brightcove 原生 SDK 可以實作，從基本玩家體驗到功能豐富的使用者體驗，包括分析、廣告等各種功能。SDK 適用於輕鬆地與 Brightcove 視頻雲和布萊特灣播放器帳戶一起工作，提供簡單的方法來傳遞您的視頻內容。布萊特灣原生 SDK 支援下列功能類別：

    播放格式和字幕

    持續掌握不斷變化的技術，對於為觀眾提供最佳的播放體驗至關重要。Brightcove SDK 致力於支援最受歡迎的影片和字幕格式，確保出版商能夠接觸世界各地的觀眾。

    學習更多關於視頻格式和字幕。

    廣告

    通過與一些線上視頻廣告中的知名商如 DoubleClick 和 Freewheel 合作，我們提供了與核心 Brightcove SDK 在幾行代碼中整合的插件。

    此外，Brightcove 原生 SDK 支援 Brightcove Once，這是我們領先業界的伺服器端廣告插入解決方案，可提供最佳化的視訊效能並防止廣告封鎖技術。

    學習更多關於使用SDK進行廣告。

    分析

    身為資料消費者，我們瞭解分析是理解內容趨勢的強大工具。無論您是使用 Brightcove Analytics 的視訊雲端客戶、使用分析 API 與您自己的後端使用分析 API 的布萊特灣播放器客戶，或是與 Omniture 等第三方整合的組合，Brightcove SDK 都能輕鬆收集資料。

    學習更多關於使用SDK進行分析。

    數字版權管理（DRM）

    我們瞭解，使用 HLSE 等標準為視訊內容提供的安全性可能不足以滿足您的嚴格保護要求。DRM 是一項強大的內容保護程式，可讓您強制執行內容使用的政策和規則。

    Brightcove SDK 可讓您整合強大的 DRM 解決方案，並確保內容的保護，而不會犧牲觀眾期待的視訊體驗。

    學習更多關於使用SDK進行數字版權管理。

    媒體內容

    作為Brightcove Video Cloud客戶，您可以通過Playback API訪問Brightcove內容（視頻，圖像資產和播放列表），CMS API或Brightcove目錄。目錄是一個助手方法庫，您可以用來訪問您的視頻雲庫。

    利用播放/CMS API 和目錄存取您的內容的工具已內建在 Brightcove 行動 SDK 的基礎上。

    只有布萊特灣視訊雲端客戶才能使用布萊特灣播放/內容管理系統 API 和布萊特灣目錄。

    現場播放

    利用 Brightcove SDK 在您的應用程式中使用即時播放的優點。啟動直播活動以提高收視率並增加興趣，或提供長時間的直播內容，例如新聞摘要和體育賽事，讓觀眾獲得最新消息和娛樂。

    Brightcove SDK 提供高品質的即時串流播放功能，讓您輕鬆設定和播放即時和 DVR 即時內容。

    學習更多關於使用SDK進行實時播放。

    Brightcove本機播放器控件

    節省開發人員成本、縮短上市時間，並保持應用程式清除不必要的程式碼雜亂。包含在我們的核心 iOS 和 Android 播放器 SDK 中，Brightcove 原生播放器控制項提供常用的使用者介面元件，以滿足您的玩家介面需求。完全可自訂，您可以變更大小、顏色，甚至新增或移除按鈕，輕鬆建立適合您應用程式的使用者介面。

    學習更多關於Brightcove本機播放器控件。

    360°視頻

    透過引人注目的 360° 視訊體驗，提升視野。這項體驗可讓觀眾沉浸在場景中，並讓觀眾在影片中捲動以互動和探索。原生 SDK 會自動偵測您的內容是否為 360° 視訊類型並播放，同時允許使用者捲動檢視。

    學習更多關於將360°視頻與本機SDK結合使用。

    多個音軌

    包含不同語言的額外音軌、評論和描述性音訊，或提供立體聲音效，以觸及更廣泛的觀眾。當您的視訊內容有多個音軌時，SDK 會自動偵測它們，並顯示音軌選取選單。

    學習更多關於本機SDK的多個音軌。

    離線播放

    Brightcove 原生 SDK 可簡化管理離線播放的複雜性，同時提供內容安全性和分析功能，提供建置此類應用程式的工具。這表示使用者可以在旅行前下載他們最愛的影片，然後在沒有網際網路連線的情況下播放。

    學習更多關於使用本機SDK進行脫機播放。
