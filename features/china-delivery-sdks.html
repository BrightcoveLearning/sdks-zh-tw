介紹

Brightcove提供了從國外向中國交付視頻的解決方案。解決方案由兩部分組成：

  • 中國境內託管的代理服務器
  • 關聯的中國特定CDN

代理將非視頻流量（播放 API 請求、分析信標和網絡播放器本身）路由到中國觀眾和來自外部的 Brightcove 基礎設施。

CDN 連接到代理並處理視頻和圖像內容。

要求

使用 Brightcove Native SDK 進行中國交付需要以下要求：

設置

  • 除了配置播放器外，還有一些設置要求

    有關詳細信息，請參見概述：中國送貨

設備操作系統版本

  • Android 6.0及更高版本

    Android 8.x 可能存在離線租賃許可問題。
  • iOS 11.0及更高版本

Brightcove SDK版本

  • 適用於Android的本機SDK（即將推出）
  • 適用於iOS 6.5.0和更高版本的Native SDK

Android實施

支持中國交付的 Android 原生 SDK 將於 2021 年第三季度初預發布。

iOS實施

要將 China Delivery 與適用於 iOS 的 Native SDK 結合使用，請按照以下步驟操作

  1. 這BCOV全局配置類允許您為 iOS 的 Native SDK 設置自定義配置。

  2. 在這個類中，設置中國代理域名播放 API、指標和分析收集服務的自定義完全限定域名 (FQDN) 的屬性。

    @property (nonatomic, nullable) NSString *chinaProxyDomain
