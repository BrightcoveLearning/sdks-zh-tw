以下是與Brightcove Native Player SDK相關的一些常見問題列表。有一個您在這裡看不到的問題嗎？讓我們知道！

一般的問題

支持隱藏式字幕嗎？

播放器 SDK 支持清單內 WebVTT(HLS) 以及 sidecar WebVTT 和 608/708 字幕。有關支持的詳細信息，請參見視頻格式和字幕文檔。

我可以自定義搜索欄嗎？

是的。如果使用Brightcove Native Player控件，則可以自定義元素，包括搜索欄。有關更多信息，請參見Brightcove本機播放器控件文檔。

我可以使用自定義控件疊加視頻嗎？

Player SDK 不會強迫您使用其控件。您可以選擇使用或不使用它們。如果使用Brightcove Native Player控件，則可以自定義元素。有關更多信息，請參見Brightcove本機播放器控件文檔。

我可以添加10秒ff / rew視頻功能嗎？

播放器 SDK 支持搜索，因此您可以實現此功能。

支持Chromecast嗎？

Brightcove Native SDK 支持將視頻流式傳輸到 Chromecast 連接的電視。
對於Android設備，請使用適用於Android的Native SDK的Cast插件。
對於iOS設備，請使用適用於iOS的原生SDK的Cast插件。

是否支持播放列表？

是的，Brightcove Native SDK 提供了從 Playback API 檢索視頻或播放列表的方法。有關詳細信息，請參見以下內容：
適用於 Android 原生 SDK 的 VideoListSampleApp
使用適用於iOS的Native SDK的播放服務檢索Brightcove資產

特定於Android

如何讓HLS在Android上運行？

Brightcove的Android播放器SDK提供了適用於Android 4.1及更高版本的HLS播放功能。

HLSe呢？

Brightcove的Android播放器SDK支持HLSe播放。

特定於iOS

是否支持AirPlay？

只要您沒有廣告，適用於 iOS 的 Player SDK 就支持 AirPlay。蘋果的AirPlay允許您無線傳輸Apple設備中的內容。

是否支持Apple TV？

適用於 iOS 的 Player SDK 的最新版本支持 Apple TV 的 tvOS 操作系統。
