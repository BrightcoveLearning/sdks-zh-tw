聲明於 BCOVCuePointProgressPolicy.h

概述

此策略為 BCOVPlayerSDK 的客戶端提供了指定響應提示點事件的代碼行為的方法。對此最常見的應用是創建廣告策略：播放頭到達指示應顯示廣告的提示點，客戶端代碼提供此策略的一個實例以指定播放器應如何行為，涉及：

  1. 哪些提示點應該由諮詢策略的代碼處理。
  2. 處理提示點後應恢復播放的時間（播放頭位置）。這是使用傳遞到策略中的提示點之一指定的，或者零（這表示內容應從其當前播放頭位置恢復）。

– 適用於事件：

返回一個BCOVCuePointProgressPolicyResult指定參考此策略的代碼將處理哪些提示點，以及在處理完所有提示點後應恢復播放的位置。

- (BCOVCuePointProgressPolicyResult *)applyToEvent:(NSDictionary *)提示點事件

參數

提示點事件

包含提示點事件信息的字典，如BCOVPlaybackController提示點委託方法。

返回值

指定提示點事件的策略結果。

討論

返回一個BCOVCuePointProgressPolicyResult指定參考此策略的代碼將處理哪些提示點，以及在處理完所有提示點後應恢復播放的位置。

此方法由提示點進度策略的使用者調用。在大多數情況下，該消費者是 Brightcove Player SDK 的廣告插件，而不是 Player SDK 的客戶端（應用程序）。如果無法使用便利工廠類方法生成策略，則自定義提示點策略應覆蓋此方法。
