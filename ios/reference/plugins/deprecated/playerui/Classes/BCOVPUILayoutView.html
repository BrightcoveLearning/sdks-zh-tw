聲明於 BCOVPUILayoutView.h

概述

BCOVPUILayoutView 視圖對像用於在BCOVPUIBasicControlView .佈局視圖通常使用 BCOVPUIBasicControlView 創建’ s +layoutViewWithControlFromTag:width:elasticity: 方法，並包含標準的播放器 UI 元素，例如播放按鈕、時間碼標籤或進度滑塊。您還可以創建空佈局視圖，在其中添加您自己的子視圖。佈局視圖屬性準確地確定了視圖在可變寬度佈局中相對於彼此的定位方式。

  最小寬度

為佈局調整大小時將為視圖設置的最小寬度。如果彈性設置為零，這將是視圖的確切寬度。

@property (nonatomic, readwrite) float minimumWidth

  彈性

彈性決定每個視圖的大小以填充播放器 UI 的可用寬度’ s 相對於所有其他視圖的控制欄。如果一個控制欄中有兩個佈局視圖，一個彈性為1.0，另一個彈性為2.0，當視圖調整大小時，後者會增長兩倍。控件佈局的每一行通常應至少包含一個彈性大於零的佈局視圖。彈性為零的視圖將保持其最小寬度。

  移除

這“移除”屬性可以設置為 YES 或 NO 以快速隱藏或顯示特定的佈局視圖。這比設置一個全新的更有效BCOVPUI控件佈局帶有刪除或添加的視圖。調用 -setNeedsLayout BCOVPUIBasicControlView設置此屬性以顯示新佈局後。
