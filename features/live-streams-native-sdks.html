介紹

Brightcove Native Player SDK的實時流支持意味著可以從任何來源快速，輕鬆地將高質量的實時廣播集成到您的應用程序中。

本機 SDK 支持直播和 DVR 直播。通過實時流媒體，原生 SDK 支持定時元數據/ID3 標籤。

使用 BrightcoveExoPlayer 視圖時，Android SDK 支持 ID3 元數據標籤。

創建實時流的一種方法是使用Brightcove Live。

有關交付實時流的選項的完整列表，請參見傳送直播文檔。

Brightcove 現場直播

Brightcove Live是創建實時流的最簡單方法。有了它，您可以交付具有可擴展性和成本效益的高清晰度 (HD) 自適應比特率實時視頻流並從中獲利。

設置您的 Brightcove Live 流，並使用適用於 Android、iOS 和 tvOS 的本機播放器 SDK 在移動設備和聯網電視上無縫播放。通過以下信息了解有關Brightcove Live的更多信息：

  • Brightcove 現場直播
  • 快速開始：Brightcove Live API

支持的功能

本機 SDK 支持的以下功能增強了實時體驗：

開箱即用的雲DVR控件

DVR Live通過允許觀看者控制其觀看體驗，超越了傳統的實時流傳輸方法。DVR Live為觀眾提供了對播放的完全控制，使您的觀眾可以在VOD風格的設置中尋找過去的時光。然後，只需單擊一個按鈕，即可無縫跳回真人秀以繼續實時觀看。

Brightcove Live服務器端廣告插入（SSAI）

服務器端廣告插入 (SSAI) 允許您將廣告拼接到服務器端的實時流中，繞過任何廣告攔截器。有關如何通過實時流設置SSAI的詳細信息，請參見Brightcove Live API：服務器端廣告插入文檔。
