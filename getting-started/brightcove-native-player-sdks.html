    布萊特灣原生 SDK

    歡迎來到布萊特灣原生 SDK 功能級別的文檔！您可以在這裡找到有關我們行動裝置和電視裝置的原生 SDK 所需的一切資訊。

    概覽

    原生 SDK 本身不是視訊播放程式。相反地，它們可以被視為與廣告、分析和數位版權管理和隱藏式字幕的整合點。

    如需原生 SDK 的快速摘要，請務必查看此影片介紹：

     

     

    原生開發可能很複雜，並且有許多方式將視訊傳入您的應用程式。如果您是該主題的新手，我們建議您閱讀有關原生環境中播放視訊的選項：

    為什麼要使用布萊特灣原生 SDK？閱讀原生視頻播放指南。

    請查看下列影片，了解如何使用原生 SDK 建立簡單的視訊應用程式：

    •  視頻培訓：使用本機SDK開發

    功能

    布萊特灣 SDK 支援一組豐富的功能，包括下列各項：

    • 字幕
    • 廣告
    • 分析
    • 內容保護
    • 即時播放
    • 標準 UI 控制項
    • 360 度視訊
    • 離線播放

    有關功能的完整列表，請參見Brightcove本機SDK功能文件。

    與功能保持最新的功能級別信息Brightcove博客。

    產品

    Brightcove 原生 SDK 專為原生平台開發人員設計，旨在透過在每個環境中遵循最新的應用程式開發標準，將上市時間降至最低。

    布賴特灣原生 SDK 建立在原生播放器框架之上。這可讓您充分利用原生作業系統的速度、效能和彈性，並將其擴充至容易整合的完整視訊體驗。此外，我們的原生 SDK 遵循已建立的記憶體、命名和設計模式慣例，讓入門變得盡可能簡單。

    原生 SDK 的設計目的是與下列類型的裝置整合：

    • 移動設備
    • 連接的電視/機頂盒

    行動裝置

    我們為移動設備提供以下 SDK，每個軟件專門用於自己的特定目的：

    • 適用於iOS的Brightcove Native SDK
    • 適用於Android的Brightcove本機SDK

    適用於 iOS 的布萊特灣原生 SDK

    適用於 iOS 的布賴特灣原生 SDK 遵循現代 iOS 慣例，因此無論您是要整合到既定的 Objective-C 應用程式或全新的 Swift 應用程式中，您都應該感到賓至如歸。

    支持的iOS版本
    為了利用Apple最新的技術改進，我們目前提供以下支持：

    • iOS 14
    • iOS 13
    • iOS 12.4.1

    在iOS 9和Xcode 7發行版中，學習如何使用名為的新功能應用程式傳輸安全性（ATS）。

    使用內部收集的指標和蘋果公共 iOS 設備使用數據報告的組合，Brightcove SDK 的 iOS 旨在接觸絕大多數用戶。

    如果您是尋求更多詳細信息的開發人員，請閱讀適用於iOS的Brightcove Native SDK技術文檔。

    適用於安卓的布萊特灣原生 SDK

    適用於 Android 的布萊特灣原生 SDK 旨在平衡達到絕大多數用戶，同時利用最新版本的 Android 版本發布的改進。

    支持的Android版本
    通過利用Google發布的改進功能來平衡這一範圍，我們目前為以下版本的Android提供支持：

    • +10.0
    • +9.0
    • 8.0, 8.1
    • 7.0, 7.1
    • +6.0

    結合使用內部收集的指標和Google的公開指標Android設備使用情況數據報告稱，針對Android的Brightcove SDK旨在吸引絕大多數用戶。

    如果您是尋找更多詳細資訊的開發人員，請閱讀適用於 Android 的 Brightcove 原生 SDK 技術文件。

    已連接的電視/機上盒

    我們提供以下 SDK 與電視機頂盒整合：

    • 適用於tvOS的Brightcove本機SDK
    • 適用於Android的Brightcove本機SDK

    適用於 tvOS 的布萊特灣原生 SDK

    透過適用於 TVOS 的布萊特灣原生 SDK，在第 4 代 Apple TV 上運用以視訊為中心的應用程式，吸引客廳觀眾！

    支持的tvOS版本
    為了利用Apple最新的技術改進，我們目前提供以下支持：

    • 14
    • 多功能產品 13
    • 電子產品 12.4.1

    進一步了解使用原生 SDK 連線的電視應用程式。

    對於想要為 Apple TV 創造引人注目的體驗的客戶而言，Brightcove 提供了一個解決方案，包括使用者體驗 (UX) 設計、高效能影片播放、易於管理的內容目錄和強大的分析功能。看看適用於Apple TV的Brightcove快速入門。

    如果您是尋求更多詳細信息的開發人員，請閱讀適用於tvOS的Brightcove本機SDK技術文檔。

    適用於安卓的布萊特灣原生 SDK

    使用 Android 版的 Brightcove 原生 SDK，擴展您的視訊內容覆蓋範圍，包括各種以 Android 為基礎的智慧型電視平台和 Fire TV。

    我們目前提供以下支援：

    • Android TV
    • 消防電視 (火力系統 5.0 以上裝置)

    進一步了解使用原生 SDK 連線的電視應用程式。

    如果您是尋找更多詳細資訊的開發人員，請閱讀適用於 Android 的 Brightcove 原生 SDK 技術文件。
