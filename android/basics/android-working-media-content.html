概述

Video Cloud 客戶可以從 Video Cloud Studio 訪問他們的媒體數據和元數據。Brightcove Player客戶將提供其媒體內容的URL。

視頻雲客戶

作為Video Cloud客戶，您可以訪問存儲在Video Cloud Studio中的媒體數據。有關更多信息，請參見適用於Android代碼示例的Player SDK。

檢索媒體數據

您可以使用 Playback API 從 Video Cloud 庫中檢索視頻和播放列表數據。有關API的詳細信息，請參見播放API概述文檔。

  1. 使用com.brightcove.player.edge.Catalog從 Brightcove 的 Playback API 檢索視頻和播放列表的類方法。您的請求可以提供視頻/播放列表ID或者ReferenceID .該服務將發出 URL 請求並解析返回的數據。

    如果您包含帶有特殊字符的ReferenceID，您必須在發出目錄請求之前對其進行 URI 編碼。

  2. 對於此請求，您將需要一個策略密鑰。如果您不熟悉策略密鑰，請參閱策略API概述文檔。

    由於本機 SDK 不發送原始標頭，因此您不能使用受域限制的策略密鑰。如果你試試這個，你會得到一個403 Forbidden當您嘗試從 Video Cloud 庫中獲取內容時的響應。

    這是有關如何使用com.brightcove.player.edge.Catalog班級：

    package com.brightcove.player.samples.exoplayer.basic;
import android.os.Bundle;
import android.util.Log;
import com.brightcove.player.edge.Catalog;
import com.brightcove.player.edge.VideoListener;
import com.brightcove.player.event.EventEmitter;
import com.brightcove.player.model.Video;
import com.brightcove.player.view.BrightcoveExoPlayerVideoView;
import com.brightcove.player.view.BrightcovePlayer;
/**
* This app illustrates how to use the ExoPlayer with the Brightcove
* Native Player SDK for Android.
*/
public class MainActivity extends BrightcovePlayer {
	private final String TAG = this.getClass().getSimpleName();
	@Override

	protected void onCreate(Bundle savedInstanceState) {
		setContentView(R.layout.activity_main);
		brightcoveVideoView = (BrightcoveExoPlayerVideoView) findViewById(R.id.brightcove_video_view);
		super.onCreate(savedInstanceState);

		// Get the event emitter from the SDK and create a catalog request to fetch a video from the
		// Brightcove Edge service, given a video id, an account id and a policy key.
		EventEmitter eventEmitter = brightcoveVideoView.getEventEmitter();
		Catalog catalog = new Catalog(eventEmitter, getString(R.string.account), getString(R.string.policy));

		catalog.findVideoByID(getString(R.string.videoId), new VideoListener() {
		  // Add the video found to the queue with add().
		  // Start playback of the video with start().
		  @Override
		  public void onVideo(Video video) {
		    Log.v(TAG, "onVideo: video = " + video);
		    brightcoveVideoView.add(video);
		    brightcoveVideoView.start();
		  }
		});
	}
}

        }

});

  2. 您可能希望查看存在的自定義字段（如果有）Video目的。將以下代碼添加到onVideo循環遍歷的回調方法customField地圖。 

    catalog.findVideoByID(getString(R.string.videoId), new VideoListener() {
   @
   Override
   public void onVideo(Video video) {
      Map<String, String> customFieldMap = (HashMap<String, String>) video.getProperties().get(Video.Fields.CUSTOM_FIELDS);
     if (customFieldMap != null && customFieldMap.size() > 0) {
       for (Map.Entry<String, String> entry : customFieldMap.entrySet()) {
         Log.v(TAG, "onVideo: Custom fields: Key: " + entry.getKey() + " Value: " + entry.getValue());
       }
     }
     brightcoveVideoView.add(video);
     brightcoveVideoView.start();
  }
});

經過地理過濾的視頻

適用於 Android 的 Brightcove Player SDK 支持地理過濾視頻。

筆記：
  • 在處理經過地理過濾的視頻時，您需要使用 Playback API。
  • 如果您的視頻在 Video Cloud 中進行了地理過濾，播放服務只會返回錯誤。如果您的視頻在 CDN 級別進行了地理過濾，則錯誤將為空。

您可以通過兩種方式向視頻添加地理過濾功能，以控制可以（或不能）在哪些國家/地區觀看視頻：

  • Brightcove的CMS API
  • Video Cloud Studio 的媒體模塊

在您的Android應用中，當您使用Brightcove的邊緣檢索視頻時目錄針對該視頻進行地理過濾的國家/地區的對象（播放API），您應該會看到以下消息：

error { message: Access to this resource is forbidden by access policy.
client_geo: us
error_subcode: CLIENT_GEO
error_code: ACCESS_DENIED }

Brightcove Player客戶

作為Brightcove Player客戶，您將提供視頻資產的URL。

對於 Android SDK Nougat (7.x)，Android 中的服務器名稱指示 (SNI) 擴展使用toASCII()對 TLS 連接的主機名中的字符實施限制的方法。如果嘗試使用不符合US_ASCII的主機名加載源（例如，主機名在子域test_example.example.com中包含下劃線），則在嘗試獲取流時會出現異常。

因此，當通過 HTTPS 加載源文件時，流的主機名必須符合 US_ASCII。

以下是將視頻添加到視頻視圖並開始播放的示例：

import com.brightcove.player.model.DeliveryType;
import com.brightcove.player.model.Video;
import com.brightcove.player.view.BrightcoveExoPlayerVideoView;
import com.brightcove.player.view.BrightcovePlayer;
import com.brightcove.player.analytics.Analytics;
@Override
protected void onCreate(Bundle savedInstanceState) {
	super.onCreate(savedInstanceState);
	setContentView(R.layout.activity_main);
	BrightcoveVideoView brightcoveVideoView = (BrightcoveVideoView) findViewById(R.id.brightcove_video_view);

	Analytics analytics = brightcoveVideoView.getAnalytics();
	analytics.setAccount("123456789");

	MediaController controller = new MediaController(this);
	brightcoveVideoView.setMediaController(controller);
	brightcoveVideoView.add(Video.createVideo("http://solutions.brightcove.com/bcls/assets/videos/Bird_Titmouse.mp4", DeliveryType.MP4));
	brightcoveVideoView.start();
}
