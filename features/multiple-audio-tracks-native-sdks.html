介紹

多個音軌可讓您提供更多信息並覆蓋更廣泛的受眾。一些示例包括不同語言的附加曲目、評論、描述性音頻和立體聲。

SDK 會自動檢測視頻在 HLS 或 DASH 清單中何時具有多個音軌。如果使用的是Brightcove默認控件，則將根據語言和變體看到唯一的音軌列表。不應該有重複。

本主題包含以下部分：

  • 創建您的視頻
  • 使用多個音軌

創建您的視頻

在我們努力將此功能引入 Video Cloud Studio 時，Video Cloud 和 Brightcove Player 客戶都需要提供他們自己的符合 HLS 和 DASH 標準的清單。

請注意，所有音軌都應具有相同的採樣率。

以下是一些有關多個音軌的有用鏈接：

  • HTML標準
  • Apple關於HLS的指南
  • DASH行業論壇

使用多個音軌

創建內容後，您可以像加載其他任何視頻一樣加載和播放它。

這是帶視頻的播放器。如果視頻準備有多個備用音軌，則耳機圖標將自動出現在控制欄中。

選擇耳機圖標後，將顯示一個音軌列表供您選擇。
