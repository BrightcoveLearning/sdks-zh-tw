介紹

將Brightcove Web Player美學與iOS和Android平台標準融合在一起，Brightcove本機播放器控件在開箱即用的一組可靠功能之上提供了時尚外觀。

Brightcove本機播放器控件的功能：

  • 動態用戶界面
  • 客制化
  • 輔助功能

動態用戶界面

Brightcove 本機播放器控件具有高度自適應性，可確保您的內容在任何情況下都顯示在漂亮的用戶界面中。

自適應屏幕尺寸

確保UI元素足夠大以清楚地看到和使用對於任何移動應用程序都是至關重要的。Brightcove 本機播放器控件包含用於確定用戶界面可用空間的邏輯。如果寬度低於某個閾值，控件將自動切換為使用兩行配置。

安卓

適用於 Android 的 Native SDK 檢查設備屏幕大小以設置播放器控件佈局。

  • 當寬度為480dp或更大，使用單行佈局。
  • 當寬度小於480dp，使用兩行緊湊佈局。

要了解有關密度無關像素 (dp) 的更多信息，請參閱 Android 的支持不同的像素密度文檔。

的iOS

適用於 iOS 的 Native SDK 檢查設備屏幕大小以設置播放器控件佈局。

  • 當寬度為450px或更大，使用單行佈局。
  • 當寬度小於450px，使用兩行緊湊佈局。

播放模式

您的播放器在其整個生命週期中可能包含不同類型的內容，每個內容都需要特定的用戶界面佈局。Brightcove Native Player Controls 包含自動確定內容類型是 VoD、Live 還是 Live DVR 並相應地調整用戶界面的邏輯，即使是在內容類型不斷變化的播放列表中！

視頻點播 (VoD)

考慮到大多數播放體驗，VoD界面是所有播放器UI設計的基礎。

Brightcove播放器控件的顯著功能包括：

  • 播放/暫停按鈕
    • 播放和暫停內容
  • 跳回按鈕
    • 按下時，使內容向後搜索 10 秒。
  • 增強的搜索欄
    • 以粉紅色顯示播放的內容
    • 顯示以淺灰色緩衝的內容
    • 支持將標記用於廣告或其他目的（Android）
  • 隱藏式字幕按鈕
    • 僅在檢測到字幕時顯示。選擇此按鈕會調出隱藏式字幕設置菜單。
  • 全屏按鈕
    • 允許觀看者進入全屏播放或返回到內聯播放
  • Airplay按鈕（僅適用於iOS）
    • 僅在檢測到Airplay設備時顯示。此功能允許觀眾設置 Airplay 流媒體。

生活

Live UI 隱藏了用戶無法參與的某些元素。刪除的元素包括“跳回按鈕"和“搜索欄"。但是，會添加一個新的“LIVE"標籤，該標籤為綠色時表示正在觀看的內容為實況。

直播DVR

Live DVR 格式提供與 VoD 佈局相同的所有功能，外加一個額外的 LIVE 標籤，當內容為直播時變為綠色，當觀眾觀看較舊的內容時變為灰色。此外，“LIVE"（實時）標籤還用作按鈕，可以隨時按下以返回到最近的實時時刻。

客制化

想要更改顏色或皮膚以匹配您應用的顏色主題嗎？沒問題！Brightcove 本機播放器控件旨在輕鬆自定義。以下是一些可以自定義控件的方法：

  • 交換顏色並使用您自己的自定義按鈕來創建個性化的主題。適用於 Android 的 Native SDK 允許您輕鬆包含/排除各種常見按鈕。在這裡，控制按鈕已更改為橙色，並且搜索欄具有自定義圖像。
  • 雖然適用於 iOS 的 Native SDK 還允許您從一組常用按鈕和佈局中進行選擇，但您也可以創建自己的自定義佈局。這是自定義佈局的示例。
  • 或者，您可以刪除按鈕以創建簡單乾淨的佈局。

輔助功能

無障礙功能是我們UI的標準組成部分，開箱即用，無需任何其他設置。

聽力障礙

Brightcove Native Player Controls 將隱藏字幕按鈕集成到每個佈局中，以確保在字幕可用時，可以輕鬆自定義字幕查看體驗。有關使用SDK的隱藏式字幕的更多信息，請參見我們的視頻格式和字幕部分。
