概述

對於Video Cloud客戶，在大多數情況下，當您使用iOS的Brightcove Player SDK時，您的應用程序會自動註冊。對於Brightcove Player客戶，您需要從您的應用程序發送您的帳戶ID。請參閱下面的詳細信息。

視頻雲客戶

如果您使用Brightcove播放器和Catalog SDK中包含的用於檢索和播放視頻的類，您的應用將自動向Brightcove註冊。此外，還將收集分析數據並將其顯示在您的視頻雲分析中。

如果您覆蓋BCOVVideo類或不使用Brightcove播放器和目錄，則需要將Video Cloud Publisher ID發送到Video Cloud Analytics。您可以通過設置分析來做到這一點account價值。這還允許您在 Video Cloud Analytics 中查看此應用程序的數據。

static NSString * const kPublisherID = @"<insert valid accountId>"; // for analytics

    _playbackController.analytics.account = kPublisherID;

有關將數據發送到 Video Cloud Analytics 的詳細信息，請參閱使用 Brightcove 分析指導。

Brightcove Player客戶

作為Brightcove Player客戶，您需要在Brightcove中註冊您的應用程序。為此，您將使用分析將您的帳戶 ID 發送到 Brightcove account價值。

static NSString * const kPublisherID = @"<insert valid accountId>"; // for analytics

    _playbackController.analytics.account = kPublisherID;
