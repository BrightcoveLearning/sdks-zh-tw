概述

當您使用適用於 tvOS 的 Brightcove Native SDK 構建 Apple TV 應用程序時，您將獲得一組播放控件，以允許用戶瀏覽視頻內容、更改視頻的縱橫比、獲取視頻信息以及管理字幕和音頻設置。

Brightcove Native SDK for tvOS 提供的控件包括以下內容：

  • 進度條也可以顯示時鐘時間
  • 視頻信息、字幕和音頻的頂部標籤欄
  • 能夠在頂部標籤欄中添加自己的自定義視圖和控件

所使用的控件和手勢對用戶來說很熟悉，因為它們與 AVKit 界面中的相似。

借助 Brightcove 服務器端廣告插入 (SSAI)，用戶將在廣告播放期間看到自定義外觀。

有關使用本機SDK的詳細信息，請參見適用於tvOS的Brightcove Player SDK概述文檔。

支持的版本

Apple TV UI控件可用於以下版本：

設備操作系統版本

tvOS 9+

本機SDK版本

適用於iOS版本6.3.0+的Brightcove本機SDK

Siri 遙控器

Siri Remote 可讓您控制 Apple TV。以下是一些可以使用它的方式：

  • 讓Siri查找您想要觀看的內容。
  • 使用觸控表面在屏幕上移動、輕掃、突出顯示和選擇項目。
  • 按下控制按鈕可快速播放/暫停內容，調節音量並導航至主屏幕和菜單屏幕。

有關更多詳細信息，請參閱Apple的將Siri Remote與Apple TV一起使用文檔。

播放控制

播放是通過Apple Siri Remote上的手勢和按鈕按下來控制的。以下是您可以與您的應用一起使用的一些操作。有關詳細信息，請參見以下內容：

  • 內置電視播放器控件，用於tvOS
  • 將 TV Player UI 與適用於 tvOS 的 Brightcove Player SDK 結合使用

進度條

輕按一下Apple Siri Remote觸控板將顯示/隱藏進度條。當播放暫停時，左右平移將在您的視頻中前後導航。

若要進行擦洗，用戶可以單擊並按住，然後向左或向右平移以在視頻中快進或快退。

在播放視頻時，用戶可以雙擊 Siri Remote 觸控板的左四分之一向後跳 10 秒，或雙擊觸控板的右四分之一向前跳 10 秒。

視頻縱橫比

雙擊Apple Siri Remote觸控板將循環顯示三種標準視頻寬高比：

  • 對寬高比表示尊重的信箱顯示（顯示完整的視頻）
  • 尊重寬高比的全屏（視頻可能會被裁剪）
  • 視圖拉伸以適合全屏（忽略縱橫比）

頂部標籤欄

在 Apple Siri Remote 觸控板上向下滑動將顯示頂部菜單並隱藏進度視圖。可見後，向上滑動將隱藏頂部菜單。

標籤欄有以下默認項目：

  • 信息
  • 字幕
  • 音訊

您可以修改與每個選項卡欄項關聯的文本標籤，也可以在選項卡欄中創建自己的自定義視圖。

信息

選擇信息頂部菜單項顯示與當前加載的視頻相關的標題、描述和長度。此數據是從 Playback API 中視頻的元數據中檢索的，但您可以在視頻加載時設置自己的文本。

字幕

選擇字幕頂部菜單項顯示用於顯示與視頻相關聯的字幕的選項。在這裡，您可以管理在播放過程中顯示的字幕。

音訊

選擇聲音的頂部菜單項顯示視頻播放期間可用的音頻選項。

風俗

借助適用於 tvOS 的 Brightcove Native SDK，您可以輕鬆地在選項卡欄中添加您自己的視圖。您的視圖可以包含您需要的任何標準 UIKit 視圖或控件。

用 SSAI 做廣告

除了內容播放之外，我們還在使用Brightcove服務器端廣告插入（SSAI）時添加了廣告模式。該模式包括以下功能：

  • 能夠防止用戶清理過去的廣告
  • 顯示有關廣告時段中剩餘時間和廣告的反饋

直播/直播DVR

使用 Apple TV 應用程序時，適用於 tvOS 的 Brightcove Native SDK 支持直播和 DVR 直播流。DVR live使用戶能夠執行以下操作：

  • 使用進度條尋找過去的時刻
  • 跳回到當前直播時刻，以實時觀看內容
