  可離線播放

確定是否可以在沒有網絡連接的情況下播放 AVAsset 的完整再現。

@property (nonatomic, readonly) BOOL playableOffline

返回值

如果此實例引用一個，則返回 YES 離線可從設備播放的視頻’■ 沒有網絡連接的本地存儲。

如果視頻尚未完成下載，或者視頻已被清除並需要重新下載，則返回 NO。如果 iOS 需要回收存儲空間以使設備正常運行，則會自動發生這種情況。

在某些版本的 iOS 上可離線播放可能返回 NO 如果離線視頻已加載到 AVPlayer 實例中。打電話用播放器物品替換當前物品在當前實例上影音播放器與零檢查之前的值應導致返回 TRUE 的預期值。

討論

在呈現或嘗試播放之前查詢此屬性離線資產以確定它是否可供使用。返回值為 true 表示此資產的完整再現可用於離線回放。

讀取這個屬性相當於讀取一個AVAsset的AVAssetCache的playableOffline屬性。
