概述

Brightcove Native SDK 包括 Android、iOS 和 tvOS。它們建立在以下框架上：

  • 安卓 - 建立在上面的圖書館Android ExoPlayer 項目 (ExoPlayer 2 v2.9.3) [1]
  • iOS /電視 -建立在Apple AVFoundation框架

除非另有說明，表中的“是”表示本機 SDK 當前支持的所有操作系統版本都支持該功能。有關受支持的操作系統版本的列表，請參閱以下內容：

  • 適用於Android的本機SDK版本
  • 適用於iOS的本機SDK版本
  • 適用於tvOS的本機SDK支持的版本
