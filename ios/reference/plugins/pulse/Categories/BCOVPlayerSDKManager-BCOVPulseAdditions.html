聲明於 BCOVPulseComponent.h

概述

類別方法添加到 BCOVPlayerSDKManager 以支持 Pulse。

–  createPulsePlaybackControllerWithPulseHost:contentMetadata:requestSettings:adContainer:companionSlots:viewStrategy:options:

使用指定的 Pulse 主機、OOContentMetadata 對象、OORequestSettings 對象和用戶選項創建並返回一個新的播放控制器。返回的播放控制器將使用 Pulse 會話提供程序進行配置。

- (id<BCOVPlaybackController>)createPulsePlaybackControllerWithPulseHost:(NSString *)脈衝主機 contentMetadata:(OOContentMetadata *)內容元數據 requestSettings:(OORequestSettings *)請求設置 廣告容器：（UIView *）廣告容器 同伴插槽：（NSArray *）同伴插槽 視圖策略：（BCOVPlaybackControllerViewStrategy）戰略 選項:(NSDictionary *)選項

參數

脈衝主機

正在使用的客戶帳戶的 Pulse 主機名。
內容元數據

一個 OOContentMetadata，用於配置返回的播放控制器使用的會話對象。
請求設置

一個 OORequestSettings，用於配置返回的播放控制器使用的會話對象。
廣告容器

將在其中顯示廣告和呈現廣告信息 UI 的視圖。
同伴插槽

要填充的 BCOVPulseCompanionSlots 數組。
戰略

一種視圖策略，用於確定返回的播放控制器的視圖。
選項

脈衝選項的 NSDictionary。可以為零或空。唯一有效的選項鍵是 kBCOVPulseOptionPulsePlaybackSessionDelegateKey kBCOVPulseOptionPulseDeviceContainerKey kBCOVPulseOptionPulsePersistentIdKey

返回值

具有指定參數的新播放控制器。

討論

使用指定的 Pulse 主機、OOContentMetadata 對象、OORequestSettings 對象和用戶選項創建並返回一個新的播放控制器。返回的播放控制器將使用 Pulse 會話提供程序進行配置。
