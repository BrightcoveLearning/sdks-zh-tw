適用於 Android 的 Brightcove Native SDK 提供了各種示例應用程序，以幫助您開始自己的實現。有關示例的完整列表，請參見以下內容：

如果您是Brightcove Native SDK的新手，請查看本機SDK入門視頻。

框架

Android 示例應用程序支持最新版本的 Brightcove Native SDK 和插件。

建議您對應用程序使用ExoPlayer 2框架，以實現Native SDK可用的最新功能。如果您當前不使用此框架，請參閱遷移到ExoPlayer 2框架文檔。

安裝示例應用

Android 示例應用程序支持最新版本的 Brightcove Native SDK 和插件。

當前，所有Android示例應用程序都必須捆綁在一起安裝。最簡單的入門方法是將示例應用導入 Android Studio。這就是我們將在以下步驟中執行的操作。

有關示例的更多詳細信息並了解安裝的版本控制選項，請參閱Android 播放器示例自述文件文件。

要導入示例應用程序，請執行以下步驟：

  1. 如果尚未安裝，請下載並安裝最新版本的Android Studio。
  2. 克隆適用於Android的播放器示例到您的計算機。
  3. 在Android Studio中，選擇導入項目。

  4. 在文件選擇對話框中，選擇頂層build.gradle本地克隆存儲庫中的文件。

    適用於 Android 示例應用程序的 Native SDK 已經使用 Gradle 構建過程進行了測試，最高版本為 3.1.4。
  5. 在Android Studio中，您應該看到所有示例應用程序項目。

運行示例應用程序

可以使用以下命令運行示例應用程序：

  • Android Studio模擬器
  • 使用連接到計算機的物理設備的Android Studio
    • 設備必須有 Android OS 4.4.4 或更高版本
    • 在您的設備上，確保已啟用USB調試

要運行示例應用程序，請執行以下步驟：

  1. 在Android Studio中，在左側項目列表中，展開BasicSampleApp項目。
  2. 展開爪哇文件夾並打開MainActivity.java文件。
  3. 在項目選擇下拉列表中，選擇BasicSampleApp。
  4. 在項目選擇下拉列表的右側，選擇跑按鈕。
  5. 如果您有連接到計算機的設備，則應該在“選擇部署目標對話。否則，在創建虛擬設備之後，您還將在此處看到它。選擇您的設備並選擇好的 .
  6. 您應該會看到該應用程序在您的設備或模擬器中運行。

調試

Android Studio有一個調試器，可讓您在運行時設置斷點，檢查變量和評估表達式。要在模擬器中調試您的應用程序，請執行以下操作：

  1. 在Android Studio中，找到您要暫停執行的代碼行。
  2. 在代碼行的左側裝訂線中，單擊以添加一個斷點。
  3. 選擇調試按鈕開始執行。

  4. 當執行在斷點處暫停時，您將有多個選項來檢查變量和表達式，以及單步執行/跳過/進入代碼行。

    有關調試應用程序的詳細信息，請參閱Android Studio 的調試用戶指南 .
