概述

縮略圖搜索允許用戶沿著時間線拖動播放頭並查看縮略圖作為相關內容的預覽。這使用戶能夠快速瀏覽視頻文件並找到他們感興趣的內容。

什麼時候生成圖像？

Brightcove可根據要求動態生成圖像。使用動態投放方式提取的任何視頻都將在第一個請求時生成圖像。這可確保不會為未使用的視頻創建圖像。生成後，就像其他任何圖像一樣，圖像會在本地和CDN上緩存以供重複查看。

視頻的圖像頻率是多少？

對於內容<1小時，每5秒鐘生成一次圖像，對於內容> 1小時，每10秒鐘生成一次圖像。

要求

使用 Brightcove Native SDK 進行縮略圖搜索需要滿足以下要求：

此功能需要使用 Playback API v2。

設備操作系統版本

  • Android 6.0以上
  • iOS 11.0以上

Brightcove本機SDK版本

  • 適用於Android 6.12.0+的本機SDK
  • 適用於iOS 6.7.4+的本機SDK

入門

任何擁有 Brightcove Video Cloud 帳戶的人都可以使用此功能。

確保為動態投放提取了您的視頻。

Android實施

要將縮略圖搜索與適用於 Android 的本機 SDK 結合使用，請執行以下步驟：

  1. 在裡面build.gradle文件，添加以下Gradle依賴項： 

    implementation "com.brightcove.player:android-thumbnail-plugin:6.13.0"

  2. 在裡面MainActivity.java文件，創建一個實例ThumbnailComponent。 

    ThumbnailComponent thumbnailComponent = new ThumbnailComponent(brightcoveVideoView);

  3. 設置預覽媒體控制器。 

    thumbnailComponent.setupPreviewThumbnailController();
  4. 有關完整的代碼示例，請參見ThumbnailScrubberSampleApp

SSAI的其他步驟

如果您使用服務器端廣告插入 (SSAI)，則還有其他步驟。

要將縮略圖插件與 SSAI 插件一起使用，您需要考慮由內容中的拼接廣告引入的偏移量。

  1. 返回您的MainActivity.java文件。
  2. 聽著SSAIEventType.AD_DATA_READY事件。
  3. 檢索SSAI視頻的時間軸對象。

  4. 創建一個實例SSAIThumbnailDocumentCreatorWrapper具有以下參數：

    • 一種ThumbnailDocumentCreator。您可以使用DefaultThumbnailDocumentCreator .
    • SSAI 視頻的時間線對象。
    • 設置ThumbnailDocumentCreator到ThumbnailPlugin .

  5. 這是一個代碼示例： 

    eventEmitter.once(AD_DATA_READY, event -> {
  Timeline timeline = event.getProperty(SSAIEvent.VMAP_TIMELINE, Timeline.class);
  ThumbnailDocumentCreator creator = new SSAIThumbnailDocumentCreatorWrapper(new   DefaultThumbnailDocumentCreator(), timeline);
  thumbnailComponent.setThumbnailDocumentCreator(creator);
});

iOS實施

要在適用於 iOS 的 Native SDK 中使用縮略圖搜索，請按照下列步驟操作

  1. 有一個新的BCOVPlaybackController啟用或禁用播放控制器的縮略圖搜索的屬性。有關詳細信息，請參見適用於iOS的本機SDK參考。

  2. 默認值為YES .要禁用縮略圖搜索，請將值設置為NO .

    /**
 * Enables or disables thumbnail seeking for this playback controller.
 *
 * Default value is YES
 */
@property (nonatomic, readwrite, assign) BOOL thumbnailScrubbingEnabled;
