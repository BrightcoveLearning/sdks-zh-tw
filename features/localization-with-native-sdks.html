概述

默認情況下，Brightcove Native SDK使用英語進行播放器控件，包括字幕選擇和廣告消息，例如“跳過廣告"，“學到更多“和 "廣告1之3"。您可以將播放器控件本地化為其他語言，如本文檔中所述。

某些語言（例如阿拉伯語）的文本是從右到左 (RTL) 讀取的。

安卓

適用於 Android 6.15.1 及更高版本的 Native SDK 支持使用以下語言本地化播放器：

  • 阿拉伯語（ar）
  • 英文（en）

當前，本機SDK僅支持從左到右（LTR）佈局進行播放和廣告控件。

使用 IMA 廣告

如果您使用的是IMA廣告，請注意IMA SDK不會自動對文本進行本地化。因此，您必須手動設置它。

  1. 首先，找到設備的當前語言：

    Locale.getDefault().getLanguage();

  2. 然後，在初始化時設置語言GoogleIMAComponent：

    ImaSdkSettings settings = ImaSdkFactory.getInstance().createImaSdkSettings();
settings.setLanguage("ar");
googleIMAComponent = new GoogleIMAComponent(
    brightcoveVideoView, 
    eventEmitter, 
    settings);

的iOS

適用於 iOS 版本 6.7.4 及更高版本的 Native SDK 支持使用以下語言本地化播放器：

  • 阿拉伯語（ar）
  • 簡體中文（zh-Hans）
  • 繁體中文（zh-Hant）
  • 英文（en）
  • 法語（法語）
  • 德文（de）
  • 日語（ja）
  • 韓語（ko）
  • 西班牙語

內置播放和廣告控件遵循現代 RTL 佈局，進度條從 LTR 移動。

使用 IMA 廣告

如果您正在使用IMA廣告，請注意Brightcove SDK不會設置該語言。因此，您必須手動設置它。

這是一個例子：

IMASettings *imaSettings = [[IMASettings alloc] init];
imaSettings.language = [NSLocale currentLocale].localeIdentifier;
