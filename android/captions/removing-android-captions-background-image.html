概述

當您將帶有字幕的視頻加載到使用適用於 Android 的 Brightcove Native Player SDK 構建的應用程序中時，媒體控制器中將出現一個字幕按鈕。

選擇媒體控制器中的字幕按鈕以顯示一個對話框設置按鈕。

選擇設置按鈕以顯示動態背景圖像以在配置標題時顯示標題。

字幕背景圖像文件大小為 400kb。如果選擇刪除該圖像，則可以用一個具有相同名稱的空圖像文件替換它。

移除背景圖片

請按照以下步驟刪除默認的字幕背景圖片：

  1. 在裡面適用於Android的Brightcove本機SDK，導航到res / drawable-nodpi / caption_background.jpg文件。
  2. 創建一個具有相同名稱的空文件， caption_background.jpg。
  3. 替換現有的caption_background.jpg文件名與您的空文件同名。
  4. 重新編譯您的項目。
