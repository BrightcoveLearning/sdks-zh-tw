概述

適用於 Android 的 Brightcove Native SDK 簡化了視頻播放與 Brightcove Video Cloud 以及許多流行的廣告、分析和 DRM 解決方案的集成。

本機 SDK 還為 KitKat 之前的設備提供隱藏式字幕支持，以及與 Brightcove Web Player 具有相似外觀和感覺的可輕鬆定制的播放器控件，以提供跨平台一致的用戶體驗。
