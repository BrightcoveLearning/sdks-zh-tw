BCOVIMAAdsRequestPolicy 類參考

繼承自 NSObject
聲明於 BCOVIMAAdsRequestPolicy.h

概述

生成 IMAAdsRequests 以供給定輸入播放會話使用的策略對象。

+   videoPropertiesVMAPAdTagUrlAdsRequestPolicy

返回一個廣告請求政策，該政策會在每個播放會話中檢查 BCOVVideo 對像以獲取要請求的 VMAP 廣告代碼 URL。如果視頻對象’ s 屬性包含一個條目，其鍵是kBCOVIMAAdTag，該條目的值假定為包含 VMAP 廣告代碼 URL 的 NSString。該 URL 將用於為該播放會話構建 VMAP IMAAdsRequest。

+ (instancetype)videoPropertiesVMAPAdTagUrlAdsRequestPolicy

返回值

廣告請求策略，可根據每個播放會話中的信息生成 VMAP IMAAdsRequests ’秒BCOVVideo.properties .

討論

返回一個廣告請求政策，該政策會在每個播放會話中檢查 BCOVVideo 對像以獲取要請求的 VMAP 廣告代碼 URL。如果視頻對象’ s 屬性包含一個條目，其鍵是kBCOVIMAAdTag，該條目的值假定為包含 VMAP 廣告代碼 URL 的 NSString。該 URL 將用於為該播放會話構建 VMAP IMAAdsRequest。

如果您希望能夠為不同的視頻指定不同的 VMAP 廣告代碼網址，請使用此政策。

+   adsRequestPolicyWithVMAPAdTagUrl：

返回一個廣告請求政策，該政策為每個播放會話生成一個帶有指定廣告代碼網址和隨播廣告位的 VMAP IMAAdsRequest。

+ (instancetype)adsRequestPolicyWithVMAPAdTagUrl:(NSString *)VMAPAdTagUrl

參數

VMAPAdTagUrl

要包含在此政策生成的 IMAAdsRequest 中的廣告代碼 URL。目前，僅支持 VMAP 廣告代碼 URL。

返回值

從指定參數生成 VMAP IMAAdsRequests 的廣告請求策略。

討論

返回一個廣告請求政策，該政策為每個播放會話生成一個帶有指定廣告代碼網址和隨播廣告位的 VMAP IMAAdsRequest。

如果您希望對所有視頻使用相同的 VMAP 廣告代碼網址，請使用此政策。

+   adsRequestPolicyWithVASTAdTagsInCuePointsAndAdsCuePointProgressPolicy：

返回檢查每個 BCOVVideo 的 BCOVCuePoints 類型的廣告請求策略‘ kBCOVIMACuePointTypeAd ’ .如果提示點對象’ s 屬性包含一個條目，其鍵是kBCOVIMAAdTag，該條目的值假定為包含 VAST 廣告代碼網址的 NSString。該 URL 將用於為該播放會話構建 VAST IMAAdsRequest。

+ (instancetype)adsRequestPolicyWithVASTAdTagsInCuePointsAndAdsCuePointProgressPolicy:(BCOVCuePointProgressPolicy *)廣告提示點進度政策

參數

廣告提示點進度政策

為 VAST 廣告指定的提示點進度政策請求提示點。如果這個參數為零， +[BCOVCuePointProgressPolicy progressPolicyProcessingCuePoints:BCOVProgressPolicyProcessAllCuePoints 恢復播放自:BCOVProgressPolicyResumeFromContentPlayhead ignoringPreviouslyProcessedCuePoints:NO]將用作默認值。

返回值

廣告請求政策，可根據每個播放視頻中的信息生成 VAST IMAAdsRequests ’秒BCOVCuePoint.properties .

討論

返回檢查每個 BCOVVideo 的 BCOVCuePoints 類型的廣告請求策略‘ kBCOVIMACuePointTypeAd ’ .如果提示點對象’ s 屬性包含一個條目，其鍵是kBCOVIMAAdTag，該條目的值假定為包含 VAST 廣告代碼網址的 NSString。該 URL 將用於為該播放會話構建 VAST IMAAdsRequest。

如果您希望能夠為不同的提示點指定不同的 VAST 廣告代碼網址，請使用此政策。

+   adsRequestPolicyFromCuePointPropertiesWithAdTag:adsCuePointProgressPolicy:

返回檢查每個 BCOVVideo 的 BCOVCuePoints 類型的廣告請求策略‘ kBCOVIMACuePointTypeAd ’ .提示點屬性將作為查詢參數附加到廣告標記上。該 URL 將用於為該播放會話構建 VAST IMAAdsRequest。

+ (instancetype)adsRequestPolicyFromCuePointPropertiesWithAdTag:(NSString *)廣告標籤 adsCuePointProgressPolicy:(BCOVCuePointProgressPolicy *)廣告提示點進度政策

參數

廣告標籤

要包含在此政策生成的 IMAAdsRequest 中的廣告代碼 URL。
廣告提示點進度政策

為 VAST 廣告指定的提示點進度政策請求提示點。如果這個參數為零， +[BCOVCuePointProgressPolicy progressPolicyProcessingCuePoints:BCOVProgressPolicyProcessAllCuePoints 恢復播放自:BCOVProgressPolicyResumeFromContentPlayhead ignoringPreviouslyProcessedCuePoints:NO]將用作默認值。

返回值

生成 VAST IMAAdsRequests 的廣告請求政策廣告標籤和每個播放視頻中的信息’秒BCOVCuePoint.properties .

討論

返回檢查每個 BCOVVideo 的 BCOVCuePoints 類型的廣告請求策略‘ kBCOVIMACuePointTypeAd ’ .提示點屬性將作為查詢參數附加到廣告標記上。該 URL 將用於為該播放會話構建 VAST IMAAdsRequest。

如果您希望對所有提示點使用相同的 VAST 廣告代碼網址，請使用此政策。
