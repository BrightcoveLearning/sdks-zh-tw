概述

Player SDK 管理器是一個單例實例，在適用於 iOS 的 Brightcove Player SDK 中充當工廠、中央註冊表和診斷中心。獲取 Manager 的實例後，使用它從 Player SDK 生態系統中獲取其他對象的實例。

向 BCOVPlayerSDKManager 添加方法以創建 FairPlay 播放器和會話提供程序。

@constant kBCOVSSTextTracksKeySourceTypeM3U8URL

  會話ID

sessionID 是與此應用程序實例關聯的唯一標識符。此標識符與其他分析數據一起發送到 Brightcove 指標服務器。

@property (nonatomic, readonly, strong) NSString *sessionID

討論

sessionID 是與此應用程序實例關聯的唯一標識符。此標識符與其他分析數據一起發送到 Brightcove 指標服務器。

sessionID 在嘗試跟踪特定播放問題時很有用。例如，如果您檢測到播放問題，您可以將 sessionID 和視頻 ID 發送到您自己的錯誤跟踪服務器，然後將這兩個值報告給 Brightcove 服務工程師，以在服務器日誌中跟踪問題的根源。
