    概述：適用於 iOS 的布萊特灣播放器 SDK

    歡迎使用適用於 iOS 開發人員的播放程式 SDK 文件。您可以在這裡找到建立原生應用程式，以便在 iOS 裝置上播放影片內容的資訊。

    文件結構

    在 Brightcove，我們努力保持文檔的結構化，乾淨和直觀。因此，我們利用 github Readme 獲取有關我們所有插件和集成的詳細信息。為了方便起見，我們已列出並連結至下列這些外掛程式。

    支援的版本

    iOS 的原生播放程式 SDK 提供下列 iOS 版本的支援：

    主動支援

    布萊特灣為下列 iOS 版本的最新 iOS SDK 提供主動支援：

    • iOS 14
    • iOS 13
    • iOS 12.4.1

    被動支援

    布萊特灣為以下 iOS 版本提供被動支援：

    • iOS 11.4.1

    瞭解支援等級

    支持的定義是[主要版本]。 [次要版本]。 [補丁版本]操作系統（例如：iOS 11.4.1）。為了充分利用 Apple 的最新技術改進，我們提供兩種等級的支援：

    • 作用中：Brightcove 將測試和修復這些操作系統版本的錯誤。
    • 被動式：Brightcove 將在這些操作系統版本上進行測試，但錯誤修復將根據個別情況進行處理。

    一般而言，Brightcove 為 80% 的使用者群所使用的作業系統版本提供主動支援，並針對至少 5% 的使用者群使用的其他版本所使用的被動式支援。

    入門

    在開始使用 iOS 的播放程式 SDK 開發之前，您應該了解以下幾個概念：

    1. 媒體

      目前版本的原生播放程式 SDK 支援透過播放 API 播放您的視訊雲端帳戶中的內容，或是做為具有網際網路存取網址的遠端資產。

    2. 分析

      透過原生播放器 SDK，視訊雲端客戶會自動取得視訊雲端中報告的強大分析資料。

    3. 核心 SDK 和相依性

      適用於 iOS 的布萊特灣播放器 SDK 及其插件可以使用 CocoapO s 依賴管理器添加到您的項目中。所有布萊特灣播放器 SDK 組件的播客規格都託管在 BrightCov特定存儲庫中。

      需要版本 1.0 或更新版本。
    4. 原生 SDK 參考

      為了幫助您了解 SDK，您可以在 iOS 版 Player SDK 中找到架構的詳細分解。瞭解關鍵元件如何互動以處理控制原生視訊播放程式的複雜性。用於iOS安裝，快速入門和體系結構的Player SDK

      在這裡，您還可以找到 SDK 的最新測試版本（如果有的話）。

    5. 本機SDK概述

      若要快速概述，請瞭解布萊特灣播放器 SDK 組件如何與原生 iOS 組件互動。iOS版Player SDK概述

    6. 問題與答案

      加入我們的 SDK 開發者論壇，其中由 Brightcove 工程師和社區解答 SDK 相關問題。搜尋先前已解答的問題或張貼新問題。身為公用群組，您可以透過提供電子郵件地址來取得存取權。

    7. 視頻培訓

      觀看一系列訓練影片，引導您完成在 Xcode 中建置 Swift 應用程式的步驟，並從您的 Video Cloud 媒體櫃播放影片。

      • Brightcove Native Player SDK入門

    以下是您在使用廣告外掛程式時可能會有幫助的其他資訊。

    IMA 外掛程式

    VAST 廣告的放置位置通常由 VMAP 文件指定。VAST 說明廣告，而 VMAP 則描述影片中廣告的位置。有關代碼的詳細信息，請參見BasicIMAPlayer iOS樣品。

    使用 IMA 廣告時，最好先檢查您的廣告是否在 SDK 以外運作，然後再實施這些廣告。以下鏈接將有助於此：

    • 數位影片多重廣告播放清單 (VMAP) 指南
    • 谷歌 IMA 技術論壇
