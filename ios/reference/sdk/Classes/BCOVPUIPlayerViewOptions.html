  學習更多按鈕瀏覽器樣式

點擊“了解更多”按鈕時用於打開 Web 視圖的樣式。默認為 BCOVPUILearnMoreButtonUseExternalBrowser。

@property (nonatomic,assign) BCOVPUILearnMoreButtonBrowserStyle learnMoreButtonBrowserStyle

討論

筆記：IMA SDK 使用自己的內部行為來顯示“了解更多”按鈕。因此，此屬性對 IMA 廣告無效。看webOpenerPresentingController在裡面IMAAds 渲染設置類以獲取更多信息。
