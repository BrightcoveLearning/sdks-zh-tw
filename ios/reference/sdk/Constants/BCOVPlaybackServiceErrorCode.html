BCOVPlaybackServiceErrorCode 常量參考

聲明於 BCOVPlaybackService.h

BCOVPlaybackServiceErrorCode

與播放 API 請求相關的錯誤。

定義

typedef NS_ENUM（NSInteger，BCOVPlaybackServiceErrorCode）{
  BCOVPlaybackServiceErrorNoError ,
  BCOVPlaybackServiceErrorCodeConnectionError ,
  BCOVPlaybackServiceErrorCodeJSONDeserializationError ,
  BCOVPlaybackServiceErrorCodeAPIError ,
};

常數

BCOVPlaybackServiceErrorNoError

沒有錯誤。

申報在BCOVPlaybackService.h .

BCOVPlaybackServiceErrorCodeConnectionError

錯誤代碼表示連接到播放 API 時出現錯誤。檢查 error.userInfo 鍵 NSUnderlyingErrorKey 以了解實際連接錯誤。

申報在BCOVPlaybackService.h .

BCOVPlaybackServiceErrorCodeJSONDeserializationError

錯誤代碼表示將響應解析為 JSON 時出錯。如果請求返回任何數據，原始 NSData 將通過鍵 kBCOVPlaybackServiceErrorKeyRawResponseData 包含在 userInfo 中。檢查 error.userInfo 鍵 NSUnderlyingErrorKey 以了解實際的 JSON 反序列化錯誤。

申報在BCOVPlaybackService.h .

BCOVPlaybackServiceErrorCodeAPIError

指示 API 返回錯誤的錯誤代碼。檢查 error.userInfo 鍵 kBCOVPlaybackServiceErrorKeyAPIErrors 以獲取 API 返回的 NSArray 錯誤，並檢查鍵 kBCOVPlaybackServiceErrorKeyAPIHTTPStatusCode 以查找返回的狀態代碼。

申報在BCOVPlaybackService.h .
