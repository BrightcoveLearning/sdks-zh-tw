概述

在“視頻雲分析"模塊中，您可以在帳戶級別或詳細級別（例如通過視頻）查看指標數據。有關在Video Cloud Analytics中查看效果報告的詳細信息，請參閱績效報告參考文檔。

以下是一些用於查看數據的選項：

  • 帳戶ID
  • 目的地
  • 播放器
  • 來源
Brightcove Analytics 將您在所有報告中設置的目標值和源值轉換為全小寫字符串值。

您還可以了解有關將數據發送到 Video Cloud Analytics .

帳戶ID

查看您從應用程序設置的源值。

accountId-視頻級別

查看您設置了 accountId 的視頻分析。

  1. 在Google Analytics（分析）模塊中，選擇表現在左側菜單中，然後選擇視頻從出現的上下文菜單中。

  2. 這其他/第三方列表項將顯示您設置了 accountId 屬性的所有視頻的性能匯總。您不會看到單獨列出的這些視頻，因為 Brightcove Analytics 目前不支持未通過 Brightcove Catalog 服務檢索的視頻的詳細報告。

目的地

查看您從應用程序設置的目標值。

目標-帳戶級別

查看您在帳戶級別設置的目標值。

  1. 在Google Analytics（分析）模塊中，選擇表現在左側菜單中，然後選擇顯示域從出現的上下文菜單中。

  2. 此帳戶績效報告列出了目標值，包括您在應用代碼中指定的值。

  3. 選擇目標域以查看詳細信息。

目的地-視頻級別

查看您在視頻級別設置的目標值。

  1. 在Google Analytics（分析）模塊中，選擇表現在左側菜單中，然後選擇視頻從出現的上下文菜單中。

  2. 選擇要觀看的視頻。

  3. 展開表現依據下拉菜單，然後選擇顯示域選項。

  4. 此視頻性能報告列出了目標值，包括您在應用代碼中指定的值。

播放器

查看您的應用程序中使用的帳戶級別的玩家。

玩家-帳戶級別

在您的應用中查看玩家的玩家分析。

  1. 在Google Analytics（分析）模塊中，選擇表現在左側菜單中，然後選擇播放器從出現的上下文菜單中。

來源

查看您從應用程序設置的源值。

來源 - 帳戶級別

查看您在帳戶級別設置的源值。

  1. 在Google Analytics（分析）模塊中，選擇表現在左側菜單中，然後選擇流量來源從出現的上下文菜單中。

  2. 在裡面概括列表中，選擇轉介鏈接以按來源查看詳細的報告。

  3. 此帳戶效果報告將包含您在應用代碼中指定的源值。

源 - 視頻級別

查看您在視頻級別設置的源值。

  1. 在Google Analytics（分析）模塊中，選擇表現在左側菜單中，然後選擇視頻從出現的上下文菜單中。

  2. 選擇要觀看的視頻。

  3. 展開表現依據下拉菜單，然後選擇流量來源選項。

  4. 此列表將顯示源的性能，包括您在應用程序代碼中指定的源值。請注意，Google Analytics（分析）模塊不支持鑽入每個列表項，例如直接的。

將數據發送到視頻雲

有關如何從您的應用程序將數據發送到Video Cloud Analytics的詳細信息，請參見以下內容：

  • 適用於iOS的Brightcove Player SDK
  • 適用於Android的Brightcove Player SDK

如果您使用SDK中包含的Brightcove播放器和目錄來檢索和播放視頻，則將自動收集分析數據並將其顯示在“視頻雲分析"模塊中。
