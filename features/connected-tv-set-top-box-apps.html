介紹

聯網電視的使用和受歡迎程度持續增長。借助 Brightcove，您可以通過大屏幕電視的高清內容與客廳裡的觀眾互動。

Native Player SDK 為自定義 UI 控件、廣告、內容保護 (DRM)、實時播放、Brightcove 分析和 QoE 指標提供無縫集成。有關這些功能的詳細信息，請參見Brightcove本機SDK功能文檔。

Brightcove使您可以使用以下平台創建引人入勝的電視應用程序：

  • 適用於tvOS的Native Player SDK
  • 適用於Android的Native Player SDK
  • 你自己的平台

適用於tvOS的Native Player SDK

借助適用於 tvOS 的 Brightcove Native Player SDK，您可以為最新一代 Apple TV 創建引人入勝的視頻體驗。Native SDK 允許您使用視頻雲庫中的內容為 Apple TV 創建以視頻為中心的應用程序，並且它支持 FairPlay Streaming (FPS) 以安全交付優質內容。

這帶有 tvOS 示例應用程序的基本 SSAI將通過為您的電視應用程序提供基礎來幫助您入門。

 

 

此示例應用程序允許您包含服務器端廣告拼接以在您的視頻內容中嵌入廣告，從而提供連續的播放流。

   

如果您是尋求更多詳細信息的開發人員，請閱讀適用於tvOS的Brightcove Native Player SDK技術文檔。

適用於Android的Native Player SDK

利用基於Brightcove的工作流程來擴大受眾範圍。借助適用於 Android 的 Brightcove Native Player SDK，您可以為基於 Android 的電視和亞馬遜的 Fire TV 創建以視頻為中心的應用程序。

適用於 Android 的原生 SDK 允許您利用來自 Video Cloud 庫的高清內容，並且支持 Widevine Modular 以安全交付優質內容。

這基本的 Android 示例應用程序將通過為您的電視應用程序提供基礎來幫助您入門。

 

 

當您準備好在您的應用程序中引入自定義控件時，請查看樣式化受控示例應用程序為指導。在此示例中，您將看到如何添加設置圖標來控製字幕顯示。

 

 

打開字幕後，您會看到它們出現在視頻底部附近。這假設您已將字幕文件與您的視頻相關聯。

 

 

如果您是尋求更多詳細信息的開發人員，請閱讀適用於Android的Brightcove Native Player SDK技術文檔。

你自己的平台

如果您想在自己的平台上開發智能電視應用，請使用Brightcove的API與您的Video Cloud庫集成。以下 API 與 Brightcove Player 和 Brightcove Native SDK 使用的 API 相同。

播放接口

使用 Playback API 從您的 Video Cloud 庫中檢索視頻和播放列表元數據。此請求還返回當前播放 URL。有關詳細信息，請參見以下內容：

  • 播放API概述

資料收集API
