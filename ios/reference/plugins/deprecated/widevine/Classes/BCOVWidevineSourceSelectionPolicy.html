BCOVWidevineSourceSelectionPolicy 類參考

繼承自 NSObject
符合 NSCopying
聲明於 BCOVWidevineSessionProvider.h

概述

Widevine 源選擇策略確定從 BCOVVideo 對像中選擇哪個源。

+   sourceSelectionWidevineWithScheme：

選擇具有指定方案的第一個 Widevine 視頻。如果沒有找到具有該方案的 Widevine 源，則返回任何方案的第一個 Widevine 源。如果沒有找到 Widevine 源，則回退到基於 HLS 方案的源選擇策略：選擇具有指定方案的 HLS 中的第一個視頻。如果未找到此類源，則返回具有任何方案的第一個 HLS 源。如果不存在 HLS 源，則返回第一個 MP4。如果以上均未找到，則返回任何類型的第一個源。

+ (BCOVBasicSessionProviderSourceSelectionPolicy)sourceSelectionWidevineWithScheme:(NSString *)方案

參數

方案

源 URL 的首選方案（kBCOVSourceURLSchemeHTTP 或 kBCOVSourceURLSchemeHTTPS）。

返回值

首選具有指定方案的 HLS 的源選擇策略。

討論

選擇具有指定方案的第一個 Widevine 視頻。如果沒有找到具有該方案的 Widevine 源，則返回任何方案的第一個 Widevine 源。如果沒有找到 Widevine 源，則回退到基於 HLS 方案的源選擇策略：選擇具有指定方案的 HLS 中的第一個視頻。如果未找到此類源，則返回具有任何方案的第一個 HLS 源。如果不存在 HLS 源，則返回第一個 MP4。如果以上均未找到，則返回任何類型的第一個源。

+   sourceSelectionWidevine

選擇第一個 Widevine 視頻，首選 HTTPS 而不是 HTTP。如果沒有找到 Widevine 源，則回退到基本源選擇策略：選擇 HLS 中的第一個視頻，無論方案如何。如果未找到 HLS 源，則選擇具有 DeliveryType 的第一個源“ MP4 ” .如果兩者都沒有找到，則返回任何類型的第一個源。這是默認的源選擇策略。
