概述

修改這些屬性後，這些更改將在下一個交付的會話中生效。要將這些屬性應用於所有會話，請在調用之前修改它們[BCOVPlaybackController setVideos:] .

  帳戶所需方法

此屬性將為 Brightcove Analytics 設置帳戶 ID 值。設置此屬性還將替換通過 Brightcove Media API 請求檢索的任何視頻的 accountId 值。

@property (nonatomic, copy) NSString *account

  目的地所需方法

此屬性必須是具有有效結構的 URI，並且必須包含授權。此屬性的默認值（如果未被覆蓋）將為“ bcsdk:// ”後跟捆綁標識符。

@property (nonatomic, copy) NSString *destination

討論

此屬性必須是具有有效結構的 URI，並且必須包含授權。此屬性的默認值（如果未被覆蓋）將為“ bcsdk:// ”後跟捆綁標識符。

請參閱http://en.wikipedia.org/wiki/URI _scheme#Generic_syntax 有關 URI 語法的更多信息和示例。

  唯一標識符已啟用所需方法

此屬性切換客戶端唯一標識符的生成。如果啟用，sdk 將使用設備識別唯一身份’■ 供應商標識符。如果禁用，sdk 將不提供唯一值，分析將依賴服務器端驅動的啟發式方法來確定唯一值。

@property (nonatomic, assign, getter=isUniqueIdentifierEnabled) BOOL uniqueIdentifierEnabled
