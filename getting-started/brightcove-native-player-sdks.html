產品

Brightcove本機SDK在設計時考慮了本機平台開發人員，旨在通過遵循每種環境中應用程序開發的最新標準來最大程度地縮短產品上市時間。

Brightcove Native SDK 建立在本機播放器框架之上。這使您可以利用原生操作系統的速度、性能和靈活性，並將其擴展為易於集成的完整視頻體驗。此外，我們的本機SDK遵循已建立的內存，命名和設計模式約定，以使入門盡可能簡單。

本機 SDK 旨在與以下類型的設備集成：

  • 移動設備
  • 連接的電視/機頂盒

移動設備

我們為移動設備提供以下 SDK，每個 SDK 專門用於其特定目的：

  • 適用於iOS的Brightcove本機SDK
  • 適用於Android的Brightcove本機SDK

適用於iOS的Brightcove本機SDK

適用於 iOS 的 Brightcove Native SDK 遵循現代 iOS 慣例，因此無論您是集成到已建立的 Objective-C 應用程序還是全新的 Swift 應用程序中，您都應該感到賓至如歸。

支持的 iOS 版本
為了利用 Apple 的最新技術改進，我們目前提供以下支持：

  • iOS 14
  • iOS 13.7
  • iOS 12.5.3

隨著 iOS 9 和 Xcode 7 的發布，了解如何使用名為應用傳輸安全 (ATS) .

結合內部收集的指標和 Apple 的公開 iOS 設備使用數據報告，適用於 iOS 的 Brightcove SDK 旨在覆蓋絕大多數用戶。

如果您是搜索更多詳細信息的開發人員，請閱讀適用於 iOS 的 Brightcove 原生 SDK技術文檔。

適用於Android的Brightcove本機SDK
