概述

適用於 iOS/tvOS 的 Brightcove Native SDK 提供了各種示例應用程序，以幫助您開始自己的實現。有關示例的完整列表，請參見以下內容：

  • iOS / tvOS的播放器示例
除了示例應用程序之外，還有一系列代碼片段可能對您有所幫助。它們可以在使用適用於 iOS 的本機 SDK 的代碼片段文檔。

如果您是Brightcove Native SDK的新手，請查看本機SDK入門視頻。

所需工具

您需要在計算機上安裝以下工具才能運行示例應用程序：

  • Xcode 8.0 或更高版本

    Xcode是一個集成開發環境 (IDE)，用於為 macOS、iOS、watchOS 和 tvOS 創建軟件。

  • CocoaPods 1.0或更高版本

    可可豆是一個依賴項管理器，用於添加Brightcove SDK框架以及其他所需的庫。您可以手動將庫添加到您的項目中，但 CocoaPods 使這變得更容易。

樣品類型

在裡面適用於iOS / tvOS的播放器示例存儲庫中，您將找到以下平台的樣本：

  • iOS-適用於Apple設備（包括iPhone，iPad和iPod）的移動操作系統

  • tvOS - Apple TV 的操作系統

示例是用以下語言編寫的：

  • 物鏡

    物鏡是C編程語言的超集，並提供面向對象的功能和動態運行時。因此，它添加了用於定義類和方法的語法。

  • 迅速

    迅速是適用於 iOS、Mac、tvOS 和 Apple Watch 的全新開源編程語言。它使用起來更直觀，類似於JavaScript。可以將其添加到現有的Objective-C代碼中。

添加庫依賴

iOS/tvOS 示例應用程序支持最新版本的 Brightcove Native SDK 和插件。

有關示例的更多詳細信息並了解安裝的版本控制選項，請參閱iOS/tvOS 播放器示例自述文件文件。

要添加核心 Brightcove SDK 以及其他庫依賴項，請執行以下步驟：

  1. 如果您還沒有可可豆，下載並安裝最新版本。
  2. 克隆或下載iOS / tvOS的播放器示例到您的計算機。
  3. 在您的計算機上，單擊您感興趣的本地樣本文件夾。在此示例中，我們將使用VideoCloudBasicPlayer樣本。
  4. 在文本編輯器中，打開VideoCloudBasicPlayer / objc / Podfile文件。請注意，沒有文件擴展名。
  5. 在Podfile中，您可以保留iOS平台版本，也可以將其更改為要在應用程序中定位的版本。
  6. 在終端應用程序中打開項目文件夾。在終端會話中，運行pod install命令。如果您發現SDK版本低於當前版本，則可以運行pod update命令以獲取最新版本。
  7. 在項目文件夾中，您現在應該看到一個.xcworkspace文件。這是您將在 Xcode 中打開的文件。

運行示例應用程序

可以使用以下命令運行示例應用程序：

  • Xcode 模擬器
  • Xcode 使用連接到計算機的物理設備
    • 設備必須有 iOS 9.0 或更高版本
    • 在您的設備上，確保已啟用USB調試

要運行示例應用程序，請執行以下步驟：

  1. 如果您不熟悉Xcode，請查看蘋果開發商文檔。
  2. 在Xcode中，打開VideoCloudBasicPlayer.xcworkspace文件。
  3. 打開ViewController.m文件。您可以使用提供的值運行應用程序，也可以將它們替換為您自己的值。在此示例中，您可以替換視頻雲帳戶ID，視頻ID和策略密鑰的值。
  4. 在工具欄的活動方案菜單中， VideoCloudBasicPlayer應該選擇項目。
  5. 展開運行目標下拉列表。如果您有連接到計算機的設備，則應該在此處看到它。選擇您的設備或模擬器版本。
  6. 選擇跑按鈕。
  7. 您應該會在您的設備或模擬器中看到該應用程序正在運行。

調試

Xcode 提供了調試工具，允許您在運行時設置斷點、檢查變量和計算表達式。要調試您的應用程序，請執行以下操作：

  1. 在Xcode中，找到要暫停執行的代碼行。
  2. 在代碼行的左側裝訂線中，單擊以添加一個斷點。雙擊以編輯斷點。右鍵單擊斷點為您提供包括刪除斷點在內的選項。
  3. 選擇跑按鈕開始執行。

  4. 當執行在斷點處暫停時，您將有多個選項來檢查變量和表達式，以及單步執行/跳過/進入代碼行。

    Xcode 提供了許多工具來幫助您測試和優化您的應用程序。有關詳細信息，請參閱Apple 的 Xcode 調試文檔。
