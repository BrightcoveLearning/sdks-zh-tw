    將 360° 視訊與原生開發套件搭配使用

    在本主題中，您將學習如何在布萊特灣原生 SDK 中使用 360° 視訊。

    簡介

    觀看 360° 影片可讓觀眾控制權，讓他們能夠探索整個場景。當影片向前移動時，觀眾可以看左、右、上、向下和後方，方法是直接在視訊檢視上平移，或是以您想要的方向移動裝置。這些身臨其境的故事使觀眾能夠成為活躍的參與者。

    本主題包含下列各節：

    • 創建和準備視頻
    • 使用Native SDK播放360°視頻
    • 適用於iOS的Native SDK
    • 適用於Android的本機SDK

    建立和準備您的視訊

    請依照下列步驟建立並準備 360° 視訊：

    1. 首先，您需要建立您的 360° 視訊。如果您是視訊雲端客戶，您也需要將其上傳至視訊雲端。要了解如何執行此操作，請參閱處理360°視頻文件。

    2. 接下來，您需要設置投影您視頻的媒體資源的價值為等矩形。這會將您的視訊識別為 360° 視訊。

      Video Cloud客戶可以使用Studio來在媒體模塊中設置投影屬性，或使用CMS API至以編程方式設置投影屬性。

      Brightcove 播放器的客戶需要手動在他們的視頻上設置這個屬性。

    3. 現在，您已準備好在應用程式中播放 360° 影片。原生 SDK 會在視訊上設定投影屬性時自動偵測 360° 視訊，並以正確的格式顯示您的視訊。

    使用原生 SDK 播放 360° 視訊

    建立內容並加入投影欄位後，就可以像其他影片一樣載入和播放。

    這裡是一個 360° 視頻的播放器。使用者可以四處瀏覽以探索整個場景。

    請注意控制列中的「360° 視訊」按鈕。點選此按鈕時，視訊會以 VR 護目鏡模式顯示並排影像。此模式只能在裝置處於橫向時使用。

    適用於 iOS 的原生 SDK

    下列資訊僅適用於 iOS 的原生 SDK。

    入門

    若要協助您開始使用 360° 視訊，請參閱下列內容：

    • 適用於iOS的Brightcove Native SDK註釋
    • 360°視頻示例應用

    方向處理

    您可以使用與其他任何影片相同的程式碼來播放 360° 視訊，但為了獲得更完整的體驗，您可能想要為控制列中的「視訊 360°」按鈕實作按鈕點選處理常式。點選此按鈕時，您可以強制將方向變為橫向。有關詳細信息，請參見360°視頻示例應用。

    功能限制

    • 使用原生 SDK 播放 360° 影片時，不支援字幕/字幕和廣告。
    • 360° 視頻使用 OpenGL 呈現視頻數據，並且由於 OpenGL 沒有實現完全加密的屏幕路徑，所以 FairPlay DRM 無法與 360° 視頻一起使用。

    適用於安卓的原生 SDK

    下列資訊僅適用於 Android 的原生 SDK。

    入門

    若要協助您開始使用 360° 視訊，請參閱下列內容：

    • 360°視頻示例應用

    功能限制

    • 使用原生 SDK 播放 360° 影片時，不支援字幕/字幕和廣告。
    • 若要播放 360° 影片，您必須使用「Exoplayer」視訊檢視。布萊特灣建議使用 Android 的原生 SDK 構建應用程序時使用 Exoplayer。有關更多詳細信息，請參見選擇視頻視圖文件。
    • 本地 SDK 目前不支持威德文 DRM。
    • 在播放清單中的 360° 和非 360 度影片之間切換時，Android 會當機。目前，您的播放清單中不能有混合的內容。
    有關使用Android Native SDK播放360°視頻的已知問題，請參見已知的問題文件。
