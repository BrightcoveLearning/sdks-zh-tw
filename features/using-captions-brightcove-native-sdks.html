概述

隱藏字幕為希望訪問它的個人提供了有關您的視頻的其他信息。字幕對於聾啞人和聽力障礙者來說是必不可少的，以訪問視頻的音頻部分。如果您不熟悉字幕，請參閱概述：向視頻添加字幕文檔。

重要的是要了解哪種字幕類型最適合每種視頻格式。首先查看視頻格式和字幕文檔。Brightcove建議使用帶有WebVTT字幕的HLS或DASH視頻。

當您使用動態攝取 API要將 SRT 或 DFXP 字幕添加到 Dynamic Delivery 視頻，字幕將自動轉換為 WebVTT 格式。了解更多關於使用 Dynamic Ingest API 添加文本軌道 .
適用於 Android 的 Native SDK 不支持 WebVTT Ruby 標籤（<ruby>和<rt>）。

字幕

對於 HLS 和 DASH 視頻，字幕播放有兩個選項：

  • 清單內WebVTT
  • 外部（Sidecar）WebVTT

清單內WebVTT

將 In-manifest WebVTT 字幕與 HLS 或 DASH 視頻一起使用時，您的字幕將正常工作。您無需執行任何額外操作即可實現這些字幕。

影片中會包含清單內的WebVTT字幕。如果將 Dynamic Ingest 與 Dynamic Delivery 結合使用，則只能創建清單內文件。有關詳細信息，請參見以下內容：

  • 提取WebVTT文件
  • 使用動態提取API提取字幕

iOS/tvOS

使用外部 WebVTT 字幕時，您需要使用邊車字幕適用於 iOS 和 tvOS 的 Player SDK 的功能。

使用BCOVPlaybackService，您的視頻和文本軌道將被格式化為 Sidecar 字幕。

對於沒有Video Cloud的Brightcove Player客戶，或者您要提供自己的視頻，則需要按照手動填充字幕數據 Sidecar字幕文檔中的“部分"。

同步 WebVTT 字幕

如果您使用的是WebVTT字幕，請確保視頻和音頻與字幕同步。了解如何在同步WebVTT字幕文檔。
