    逐步：使用目標 C 的簡單視頻應用程序

    在本主題中，您將學習如何使用適用於 iOS 的 Brightcove 播放器 SDK 來建置簡單的視訊播放應用程式。您將使用 Objective-C 編程語言構建它。

    概覽

    此範例會播放從 Brightcove 播放 API 傳回的視訊，這是最新且建議的 API，可從您的視訊雲端庫擷取內容。

    對於此選項，您將需要原則金鑰。如果您不熟悉原則金鑰，請參閱原則 API 概觀。

    這個示例是用Objective-C編寫的，但是您也可以嘗試使用Swift。請參閱以下內容：
    • 一步步：：使用 Swift 的簡單視頻應用
    • Swift版本的VideoCloudBasicPlayer

    開始使用

    此範例會從內部視訊檔案陣列中播放視訊。

    請依照下列步驟熟悉設定使用 iOS 版 Brightcove 播放程式 SDK 的應用程式專案。您可以檢視下列各項的完整程式碼：

    • 應用程式代表實作
    • 檢視控制器標頭

    • View Controller實施

    有兩種方法可以嘗試這個範例：

    下載範例

    下載整個 Xcode 項目進行實驗。

    1. 將 iOS 版本的原生 SDK 複製或下載至您的本機系統。
    2. 導航到播放器/ VideoCloudBasicPlayer / objc示例應用程序。
    3. 執行pod install指令。
    4. 開啟並執行新建立的工作區。

    如需詳細資訊，請參閱適用於 IOS/TSOS 的原生 SDK 執行範例應用程式文件。

    依照本指南中的步驟建置應用程式

    請依照下列步驟熟悉設定使用適用於 tvOS 的 Brightcove 播放程式 SDK 的應用程式專案。您可以檢視下列各項的完整程式碼：

    1. 應用程式代表實作
    2. 檢視控制器標頭
    3. View Controller實施

    建立專案

    在 Xcode 中設置一個項目。然後將Brightcove Player SDK以及所有依賴項添加到項目中。

    設置 Xcode 項目

    為應用程序創建一個新的 Xcode 項目。

    1. 在 Xcode 中開始創建一個新的 iOS 項目。選擇創建一個新的 Xcode 項目。或者，在Xcode頂部菜單中，您可以選擇文件，新，項目。

    2. 選擇單視圖應用程序為模板，然後選擇下一個。

    3. 設定專案資訊，如下所示：

      • 產品名稱：簡單影片播放
      • 群組：無

        [專案團隊] 欄位是選擇性的。如果你想在 App Store 發佈你的應用程式，你需要透過 Apple 開發者計劃選擇你屬於的團隊。程式碼簽署需要群組名稱。在此範例中，我們將選取「無」。

      • 組織名稱：您的公司名稱
      • 組織識別碼： com.examplemp-公司
        這使得應用程式商店的產品名稱唯一。

    4. 選擇您要儲存專案並建立專案的位置。

    5. 現在關閉項目（是的，關閉它-這很重要！）

    將 SDK 及其依賴項添加到您的項目

    將 SDK 及其依賴項添加到您的項目中的最簡單方法是使用 Cocoapods。

    Cocoapods 是一個依賴管理器，將庫添加到您的項目中。它不是必需的，但使安裝更容易。若要安裝 Coapods，請參閱 Cocoapods 網站上的說明。

    • 需要版本 1.0 或更新版本。
    • 在現有的 Xcode 項目上使用 Cocoapods（該pod工具）將修改項目文件。您可能需要在執行此操作之前進行備份。

    1. 在您的項目文件夾中，創建一個名為 Podfile （無文件擴展名）的純文本文件。

    2. 使用文本編輯器，將以下代碼行添加到 Podfile 並保存它。此程式碼會執行下列作業：

      • 第 1 行：指向 GitHub 位置的布萊特灣網繭規格。
      • 第 3 行：將 Pods 設置為使用框架而不是靜態庫。
      • 第 5 行：定義 iOS 平台版本。

      • 第 7-9 行：安裝布萊特灣原生播放器 SDK。

        1. 從 iOS 概述文檔的布萊特灣原生 SDK 獲取當前的原生 SDK 版本。
        2. 將值 '6.3.11'取代為目前的 SDK 版本號碼。

        3. 如需有關管理相依性的詳細資訊，請參閱如何使用 Cocoapods 管理相依性文章。

         
        該target屬性用於可卡普 1.0 版和更新版本。
      source 'https://github.com/brightcove/BrightcoveSpecs.git'

use_frameworks!

platform :ios, '11.0' # (or whatever version you want)

target 'Simple-Video-Playback' do
  pod 'Brightcove-Player-Core', '6.3.11'
end

    3. 打開終端會話並導航到您的簡單視頻播放 Xcode 項目文件夾。

    4. 在終端會話中，鍵入以下命令：

      pod install

      ，然後按 Return 鍵執行它。您應該會在終端中看到一系列訊息，指出 Brightcove 播放程式 SDK 已新增至您的專案。

    5. 請注意最後一行，這很重要-從這一點開始，您必須在 Xcode 中打開簡單視頻播放 .xcWorkspace 文件，而不是簡單的視頻播放 .xCodeproj 文件。指pod install令會為您的專.xcworkspace案建立此檔案。

    • 如果您定期建立和編輯 Podfile，您可以嘗試使用 Cocoapods 應用程式來完成上述步驟。
    • 如果您不想使用 Cocoapods，您可以下載最新版本的 SDK 並手動將其添加到您的項目中。

    設定應用程式的音訊行為

    音訊工作階段會處理應用程式層級的音訊行為。進一步了解音訊會議課程。

    您可以從數個音訊工作階段類別和設定中選擇，以自訂應用程式的音訊行為。為您的應用程式選擇最佳的音訊工作階段類別。

    對於此示例，我們將使用AVAudioSessionCategoryPlayback。即使螢幕鎖定且「鈴聲/靜音」開關設定為靜音，也會播放音訊。為簡單起見，我們將為此代碼放入App Delegate中。

    1. 在您的項目中，打開App Delegate實施文件（ AppDelegate.m）。

    2. 在裡面didFinishLaunchingWithOptions方法，添加代碼以設置音頻會話類別。確保您導入AVFoundation框架。

      • 為了簡化示例，我們將其放入應用程序委託中。有關如何在自己的應用中使用此功能的更多信息，請參見音頻會話編程指南。

{
