介紹

借助分析數據，您可以跟踪視頻的成功情況並更好地了解觀眾的習慣。適用於 iOS 和 Android 的 Brightcove Native Player SDK 可以輕鬆地將分析集合集成到您的視頻解決方案中。

分析解決方案：

  • 視頻雲分析
  • Adobe Analytics（辦公用品）

視頻雲分析

使用目錄服務的 Video Cloud 客戶將擁有由我們自己的 Brightcove Analytics 系統自動收集的分析數據。然後將這些數據匯總並顯示在視頻雲分析模塊中。下表突出顯示了當前在 SDK 中收集的數據點：

除了這些事件之外，您還可以指定自己的自定義數據（可在應用程序中配置），該數據可以發送和匯總以供Brightcove Analytics查看。

Brightcove Analytics僅適用於Brightcove Video Cloud客戶。

安卓

有關將Omniture插件添加到您的應用程序的詳細信息，請按照下列步驟操作：

  1. 從基本 Omniture 示例應用程序 .

  2. 在裡面app / res / layout / omniture_activity_main.xml文件，替換以下內容

    <com.brightcove.player.view.BrightcoveVideoView

    有了這個

    <com.brightcove.player.view.BrightcoveExoPlayerVideoView

  3. 在裡面build.gradle，找到Omniture代碼塊並將其替換為：

    if (project.name.contains('Omniture')) {
  dependencies {
    implementation "com.brightcove.player:android-omniture-plugin:${anpVersion}"
    implementation files(libraryDirectory + "adobeMobileLibrary.jar")
    implementation "com.brightcove.player:exoplayer2:${anpVersion}"
  }
}

  4. 在裡面MainActivity.java文件，找到onCreate()功能。哪裡BrightcoveVideoView已創建，將其替換為：

    brightcoveVideoView = (BrightcoveExoPlayerVideoView) findViewById(R.id.brightcove_video_view);
  5. 您已準備好使用 ExoPlayer 框架運行示例應用程序。
