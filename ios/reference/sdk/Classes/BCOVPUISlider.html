概述

UISlider 的一個子類，它提供了一個“緩衝的”帶有用於廣告或一般用途的標記的軌道，加上一個“廣告模式”用於顯示廣告的進度。

  廣告模式

設置為 YES 以更改滑塊’ s UI 到廣告模式。在廣告模式下，這會隱藏緩衝指示器、隱藏標記並將最小/最大軌道顏色設置為廣告MinimumTrackTintColor和廣告MaximumTrackTintColor .

–  addMarkerAt:duration:isAd:image:

添加將顯示在滑塊上特定位置的標記。

- (void)addMarkerAt:(double)位置 持續時間：（雙倍）期間 isAd:(BOOL)廣告 圖像:(UIImage *)圖片

參數

位置

滑塊上應放置標記的位置（以秒為單位）。見“期間”相關細節的財產。
期間

這期間由刻度線表示；可以設置為零以繪製最小尺寸的標記（2 像素寬）。
廣告

YES 表示此標記代表一個廣告（默認情況下以黃色繪製）。NO 表示這是一個通用標記（默認為白色）。
圖片

將代替默認標記指示器繪製的可選圖像。可以設置為零。

討論

添加將顯示在滑塊上特定位置的標記。

–  setCustomMinimumTrackImage:forState:

設置在正常（非廣告）模式下使用的自定義最小軌道圖像。

- (void)setCustomMinimumTrackImage:(UIImage *)圖片 forState:(UIControlState)狀態

參數

圖片

要在軌道的最小端（通常是拇指的左側）上使用的圖像。圖像應該是一個可調整大小的 UIImage，‘軌道高度’像素高。設置自定義圖像後，將不再顯示標記。設置為 nil 以恢復默認軌道圖像。
狀態

此圖像將為其激活的 UIControlState。

討論

設置在正常（非廣告）模式下使用的自定義最小軌道圖像。
