概述

Native Player SDK 提供類似於 Brightcove Web Player 的播放器控件，以提供跨平台一致的用戶體驗。您可以使用開箱即用的 Brightcove 播放器控件，或使用您自己的顏色、樣式和按鈕對其進行自定義。

筆記：

要了解有關這些功能的更多信息，請參閱Brightcove 本機播放器控件文檔。

有關適用於Android的本地SDK的屬性值的完整列表，請參見瀏覽SDK屬性值本文檔的第二部分。

默認播放器控件

當您使用適用於 Android 的 Brightcove Player SDK 創建應用程序時，您應該從BrightcovePlayer或者BrightcovePlayerFragment班級。

擴展BrightcovePlayer

在您的應用代碼中，擴展MainActivity上一類中的上一類。例如：

public class MainActivity extends BrightcovePlayer {
}

現在，您可以在代碼中實例化視頻視圖了。

實例化BrightcoveVideoView

在裡面MainActivity.java文件，然後在onCreate()方法，添加以下代碼：

@Override
protected void onCreate(Bundle savedInstanceState) {
    setContentView(R.layout.activity_main);
    brightcoveVideoView = (BrightcoveExoPlayerVideoView) findViewById(R.id.brightcove_video_view);
    super.onCreate(savedInstanceState);
}

自定義播放器控件

Brightcove 播放器控件支持字幕、視頻倒帶和全屏模式。您還可以選擇用於顯示和隱藏控件的動畫樣式。

更改控件顏色

可以輕鬆自定義，您可以更改Brightcove播放器控件的顏色以匹配您公司的品牌材料。

  1. 在您的代碼中使用相同的代碼主活動.java文件如上。
  2. 在你的src / main / res / values / colors.xml或者res / values / strings.xml文件，包括以下代碼： 
    <?xml version="1.0" encoding="utf-8"?>
<resources>

    <!-- Define the new colors. -->
    <color name="bc_orange">#F3951D</color>
    <color name="magenta_semi_trans">#4DFF00FF</color>

    <!-- Change the progress colors to magenta/black/orange, all transparent. -->
    <color name="bmc_seekbar_played">@color/magenta_semi_trans</color>
    <color name="bmc_seekbar_buffered">@color/black_semi_trans</color>
    <color name="bmc_seekbar_track">@color/bc_orange</color>

</resources>

更改播放器控件中的圖標和按鈕

要進一步自定義 Brightcove 播放器控件，您可以更換圖標並添加自己的按鈕以匹配您的公司風格。

  1. 在您的代碼中使用相同的代碼主活動.java文件如上。
  2. 在你的src / main / res / values / strings.xml文件，包括以下代碼： 
    <?xml version="1.0" encoding="utf-8"?>
<resources>
    <!-- Use the Font Awesome fa-stepbackward icon for the rewind
         button face. -->
    <string name="brightcove_controls_rewind">&#xf048;</string>
</resources>

顯示/隱藏播放器控件

在視頻播放過程中顯示和隱藏播放器控件的方式有以下三種：

  1. 用戶可以點擊設備屏幕以在顯示和隱藏控件之間切換播放器控件。

  2. 您可以使用BrightcoveMediaController班級。這是一個如何顯示已定義的播放頭位置範圍的播放器控件的示例。

    brightcoveVideoView.getEventEmitter().on(EventType.PROGRESS, new EventListener() {
    @Override
    public void processEvent(Event event) {
        int playheadPosition = brightcoveVideoView.getCurrentPosition();
        if(playheadPosition > 24000 && playheadPosition < 29000) {
            BrightcoveMediaController controller = brightcoveVideoView.getBrightcoveMediaController();
            controller.show();
        }
    }
});

    如果您嘗試調用getBrightcoveMediaController()加載視頻之前的方法。您還可以收聽DID_SET_VIDEO調用此方法之前的事件。

    請注意，播放器控件將在您顯示 3 秒後自動隱藏，除非您覆蓋此默認行為。

  3. 您可以按如下方式使用事件：

    import static com.brightcove.player.mediacontroller.ShowHideController.SHOW_MEDIA_CONTROLS;
import static com.brightcove.player.mediacontroller.ShowHideController.HIDE_MEDIA_CONTROLS;
...
eventEmitter.emit(HIDE_MEDIA_CONTROLS);
eventEmitter.emit(SHOW_MEDIA_CONTROLS);

禁用播放器控件

在設計應用佈局和功能時，您可能需要關閉播放器控件。這可以通過設置MediaController歸零如下：

@
Override
public View onCreateView(LayoutInflater inflater, ViewGroup container, Bundle savedInstanceState) {
   View rootView = inflater.inflate(R.layout.fragment_player, container, false);
   brightcoveVideoView = (BrightcoveExoPlayerVideoView) rootView.findViewById(R.id.brightcove_video_view);
   super.onCreateView(inflater, container, savedInstanceState); // Need to be called after brightcoveVideoView is set.
   brightcoveVideoView.setMediaController((MediaController) null); // Need to be called after onCreateView.
   return rootView;
}

這個例子使用了BrightcovePlayerFragment .

這BrightcoveMediaController類提供默認的 Brightcove 播放器控件並允許您管理和自定義它們，包括控制欄、搜索欄和一組媒體控制按鈕。您還可以禁用 Brightcove Player 控件以使用 Android 原生控件。

手動控制全屏模式

通常，用戶將通過點擊控制欄中的全屏按鈕來控制全屏查看。有時您可能希望在代碼中手動控制此功能。

要設置全屏模式，請使用以下代碼：

brightcoveVideoView.getEventEmitter().emit(EventType.ENTER_FULL_SCREEN);

要退出全屏模式，請使用以下代碼：

brightcoveVideoView.getEventEmitter().emit(EventType.EXIT_FULL_SCREEN);

有關事件類型的完整列表，請參見適用於Android的Player SDK，然後導航到事件類型班級文件。

或者，您可以使用fullScreen()和normalScreen()方法。你可以在這些方法中找到適用於 Android 的播放器 SDK 參考，並導航到BrightcovePlayer類文件。

瀏覽SDK屬性值

要獲取完整的值列表，包括數組，顏色，尺寸，id，整數，字符串和样式，請執行以下操作：

  1. 找到android-native-player-public-xxxzip文件並解壓縮。

  2. 在裡面sdk文件夾，打開android-sdk-xxxaar文件。

    一種打開方式阿爾文件是將擴展名從阿爾到壓縮並解壓。
  3. 打開res / values / values.xml文件。在這裡，您將找到可以在應用程序中引用的值的列表。
