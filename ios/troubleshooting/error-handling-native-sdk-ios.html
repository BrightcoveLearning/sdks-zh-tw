概述

錯誤代碼包含以下術語ErrorCode在名字裡。例如：

kBCOVPlaybackSessionErrorCodeLoadFailed

活動有一個詞Event在名字裡。例如：

kBCOVPlaybackSessionLifecycleEventFail

播放錯誤

播放錯誤是通過事件處理的。一般而言，您會在控制台上使用生命週期事件委託回調收到播放錯誤BCOVPlaybackController：

- (void)playbackSession:(id<BCOVPlaybackSession>)session
  didReceiveLifecycleEvent:(BCOVPlaybackSessionLifecycleEvent *)lifecycleEvent

要獲取更多信息，請執行以下操作：

  1. 如果您收到類似的錯誤事件kBCOVPlaybackSessionLifecycleEventFail或者kBCOVPlaybackSessionLifecycleEventResumeFail，檢查事件的properties字典@"error"鑰匙。

    這將提供一個NSError帶有問題信息的對象。

  2. 每個NSError將具有錯誤域，如下所示：

    • NSCocoaErrorDomain
    • kBCOVPlaybackSessionErrorDomain
     

    每個錯誤域都將具有關聯的錯誤代碼和錯誤描述，如下所示：

    • kBCOVPlaybackSessionErrorCodeLoadFailed
    • kBCOVPlaybackSessionErrorCodeNoPlayableSource

  3. 一些錯誤還會有一個潛在的錯誤，可以從錯誤對象的userInfo字典與NSUnderlyingErrorKey鑰匙。這通常是一個系統NSError可以提供更多信息。

  4. 要查找 SDK 中的所有錯誤域，請在標頭中搜索以下內容：

    ErrorDomain

    這是具有以下內容的標題列表ErrorDomain參考：

    • BCOVFPSAuthorizationProxy.h
    • BCOVFPSBrightcoveAuthProxy.h
    • BCOVOfflineVideoManager.h
    • BCOVOnceCommon.h
    • BCOVOnceONOXMLDocument.h
    • BCOVOUXConstants.h
    • BCOVPlaybackService.h
    • BCOVPlaybackSession.h

  5. 要查找所有錯誤代碼，請搜索每個ErrorDomain對於以下情況：

    ErrorCode

    例如，在BCOVPlaybackSession.h， 你會看到的kBCOVPlaybackSessionErrorDomain帶有以下錯誤代碼：

    • kBCOVPlaybackSessionErrorCodeLoadFailed
    • kBCOVPlaybackSessionErrorCodeFailedToPlayToEnd
    • kBCOVPlaybackSessionErrorCodeNoPlayableSource

  6. 其他域可能會以不同的方式返回錯誤。例如，BCOVPlaybackService如果從在線目錄中請求視頻時出現網絡錯誤，則該對象將在其完成塊中返回錯誤。它使用錯誤域kBCOVPlaybackServiceErrorDomain。
