播放格式和字幕

掌握不斷變化的技術對於為觀眾提供最佳播放體驗至關重要。Brightcove SDK 努力支持最流行的視頻和字幕格式，以確保發布商能夠接觸到世界各地的觀眾。

學習更多關於視頻格式和字幕。

廣告

通過與 DoubleClick 和 FreeWheel 等在線視頻廣告領域的一些知名企業合作，我們提供的插件只需幾行代碼即可與核心 Brightcove SDK 集成。

此外，Brightcove Native SDK支持我們行業領先的服務器端廣告插入解決方案Brightcove Once，可提供優化的視頻性能並防止廣告攔截技術。

學習更多關於使用SDK進行廣告。

分析

作為數據消費者自己，我們認識到分析是了解您內容趨勢的強大工具。無論您是使用 Brightcove Analytics 的視頻雲客戶、使用帶有自己後端的分析 API 的 Brightcove Player 客戶，還是與第三方集成（如 Omniture）的某種組合，Brightcove SDK 都可以讓您輕鬆收集數據。

學習更多關於使用SDK進行分析。

數字版權管理（DRM）

我們認識到，使用 HLSe 等標準為視頻內容提供的安全性可能不足以滿足您嚴格的保護要求。DRM是一個強大的內容保護程序，可讓您強制執行政策和使用內容的規則。

Brightcove SDK 使您能夠集成強大的 DRM 解決方案，並確保在不犧牲觀眾期望的視頻體驗的情況下保護您的內容。

學習更多關於使用SDK進行數字版權管理。

媒體內容

作為 Brightcove Video Cloud 客戶，您可以通過 Playback API 訪問您的 Brightcove 內容（視頻、圖像資產和播放列表），內容管理系統 API或 Brightcove 目錄。該目錄是一個可用於訪問 Video Cloud 庫的輔助方法庫。

通過播放/CMS API 和目錄訪問您的內容的工具內置於 Brightcove Mobile SDK 的基礎中。

只有 Brightcove Video Cloud 客戶可以訪問 Brightcove Playback/CMS API 和 Brightcove 目錄。

現場回放

使用 Brightcove SDK 在您的應用程序中利用實時回放的優勢。發起直播活動以提高觀眾人數並產生興趣，或者提供長途直播內容（例如新聞源和體育賽事），以使觀眾保持知情和娛樂。

Brightcove SDK支持實時流的高質量播放，使您可以輕鬆配置和播放Live和DVR Live內容。

學習更多關於使用SDK進行實時播放。

Brightcove 本機播放器控件

節省開發人員成本、縮短上市時間並使您的應用程序遠離不必要的代碼混亂。Brightcove本機播放器控件包含在我們的核心iOS和Android播放器SDK中，提供了通用的用戶界面組件，可以滿足您的播放器界面需求。完全可定制的，您可以更改大小，顏色，甚至添加或刪除按鈕，以輕鬆構建適合您的應用程序的用戶界面。

學習更多關於Brightcove本機播放器控件。

360°視頻

引人入勝的360°視頻體驗帶動觀看。這種體驗讓觀眾沉浸在場景中，並允許他們通過滾動視頻進行交互和探索。Native SDK 會自動檢測您的內容是否為 360° 視頻類型並播放它，同時允許用戶滾動視圖。

學習更多關於將360°視頻與本機SDK結合使用。

多個音軌

通過包括其他語言，註釋和描述性音頻或提供立體聲的其他音頻軌道，可以吸引更廣泛的受眾。當您的視頻內容有多個音軌時，SDK 會自動檢測它們並顯示音軌選擇菜單。

學習更多關於本機SDK的多個音軌。

離線播放

Brightcove Native SDK 簡化了管理離線播放的複雜性，同時還提供內容安全和分析，提供了構建此類應用程序的工具。這意味著用戶可以在旅行前下載他們最喜歡的視頻，然後在沒有互聯網連接的情況下播放。

學習更多關於使用本機SDK進行脫機播放。
