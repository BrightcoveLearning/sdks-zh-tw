介紹

觀看 360° 視頻將控制權交還給觀看者，以便他們可以探索整個場景。當視頻向前移動時，觀看者可以向左、向右、向上、向下和向後看，方法是用手指直接在視頻視圖上平移，或將設備移動到您想要看到的方向。這些身臨其境的故事使觀眾能夠成為行動的積極參與者。

本主題包含以下部分：

  • 創建和準備視頻
  • 使用Native SDK播放360°視頻
  • 適用於 iOS 的原生 SDK
  • 適用於 Android 的原生 SDK

創建和準備視頻

請按照以下步驟創建和準備您的視頻360°視頻：

  1. 首先，您需要創建360°視頻。如果您是Video Cloud客戶，則還需要將其上傳到Video Cloud。要了解如何執行此操作，請參閱處理 360° 視頻文檔。

  2. 接下來，您需要設置投影您視頻的媒體資源的價值為等矩形。這會將您的視頻標識為 360° 視頻。

    Video Cloud 客戶可以使用 Studio 來在媒體模塊中設置投影屬性，或使用內容管理系統 API到以編程方式設置投影屬性。

    Brightcove Player客戶需要在其視頻上手動設置此屬性。

  3. 現在，您可以在應用中播放360°視頻了。當在視頻上設置了投影屬性並以正確的格式顯示您的視頻時，本機 SDK 會自動檢測 360° 視頻。

使用Native SDK播放360°視頻

創建內容並添加投影字段後，您就可以像加載其他任何視頻一樣加載和播放它。

這是具有360°視頻的播放器。用戶可以四處導航以探索整個場景。

請注意控制欄中的“視頻360°"按鈕。當點擊此按鈕時，視頻以 VR Goggles 模式顯示並顯示並排圖像。此模式只能在設備處於橫向時使用。

適用於 iOS 的原生 SDK

以下信息特定於適用於 iOS 的 Native SDK。

入門

為了幫助您開始使用 360° 視頻，請參閱以下內容：

  • 適用於iOS的Brightcove Native SDK註釋
  • 視頻 360° 示例應用程序

方向處理

您可以使用與播放任何其他視頻相同的代碼來播放 360° 視頻，但為了獲得更完整的體驗，您可能需要為控制欄中的 Video 360° 按鈕實現一個按鈕點擊處理程序。當點擊此按鈕時，您可以將方向強制為橫向。有關詳細信息，請參見視頻360°示例應用。

功能限制

  • 使用Native SDK播放360°視頻時，不支持字幕/字幕和廣告。
  • Video 360° 使用 OpenGL 來呈現視頻數據，由於 OpenGL 沒有實現到屏幕的完全加密路徑，FairPlay DRM 不能用於 360° 視頻。
  • 雖然 360° 直播流可以與 Native SDK 一起使用，但它不受官方支持。

適用於 Android 的原生 SDK

以下信息特定於適用於 Android 的本機 SDK。

入門

為了幫助您開始使用 360° 視頻，請參閱以下內容：

  • 視頻 360° 示例應用程序

功能限制

  • 使用Native SDK播放360°視頻時，不支持字幕/字幕和廣告。
  • 要播放 360° 視頻，您必須使用 ExoPlayer 視頻視圖。在使用適用於Android的Native SDK構建應用程序時，Brightcove建議使用ExoPlayer。有關更多詳細信息，請參見選擇視頻視圖文檔。
  • 適用於 Android 的本機 SDK 當前不支持 Widevine DRM。
  • 在播放列表中的360°和非360°視頻之間切換時，Android崩潰。當前，您的播放列表中不能包含混合內容。
  • 雖然 360° 直播流可以與 Native SDK 一起使用，但它不受官方支持。
有關使用適用於 Android 的本機 SDK 播放 360° 視頻的已知問題，請參閱已知的問題文檔。
