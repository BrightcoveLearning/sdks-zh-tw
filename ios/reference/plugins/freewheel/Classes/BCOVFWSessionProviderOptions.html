概述

FW 會話提供程序的可選配置。

  等待AVPlayer持續時間

在調用 adContextPolicy 之前是否等待 AVPlayer 項報告持續時間。當設置為 NO 時，持續時間傳遞到BCOVFWSessionProviderAdContextPolicy塊可能不准確。默認為是。

@property (nonatomic, assign, getter=shouldWaitForAVPlayerDuration) BOOL waitForAVPlayerDuration

討論

在調用 adContextPolicy 之前是否等待 AVPlayer 項報告持續時間。當設置為 NO 時，持續時間傳遞到BCOVFWSessionProviderAdContextPolicy塊可能不准確。默認為是。

  廣告超時間隔

會話提供程序在不使用廣告的情況下提供會話之前等待廣告請求的秒數。默認為 3 秒。

@property (nonatomic, assign) NSTimeInterval adTimeoutInterval

討論

會話提供程序在不使用廣告的情況下提供會話之前等待廣告請求的秒數。默認為 3 秒。

  提示點進度策略

一種策略，用於確定在給定提示點事件發生時是否應播放 FreeWheel 廣告，如果是，應從何處恢復播放。默認為 BCOVProgressPolicyProcessFinalCuePoint、BCOVProgressPolicyResumeFromContentPlayhead、YES。

@property (nonatomic, strong) BCOVCuePointProgressPolicy *cuePointProgressPolicy

  預載插槽

確定插槽是否已預加載，以便它們在被擊中時準備好播放。當設置為 YES 時，預卷插槽將盡快加載。Midrolls 插槽將在播放插槽前 X 秒加載，由廣告到預加載時間 .默認為否。

@property (nonatomic, assign, getter=shouldPreloadSlots) BOOL preloadSlots

討論

確定插槽是否已預加載，以便它們在被擊中時準備好播放。當設置為 YES 時，預卷插槽將盡快加載。Midrolls 插槽將在播放插槽前 X 秒加載，由廣告到預加載時間 .默認為否。

@property (nonatomic, assign) NSTimeInterval timeBeforeAdToPreload
